# 天命共振 - 虛擬人物紙娃娃系統視覺規格書

**作者**：天命美術 Agent
**日期**：2026-03-23
**對象**：遊戲 Agent、系統 Agent

.

本文件旨在說明「天命共振」虛擬人物紙娃娃系統的視覺架構、圖層規範與 AI 生成可行性評估，作為後續遊戲化模組開發的基礎參考。

.

## 1. 視覺風格定調

為符合「天命共振」18-35 歲目標受眾的審美，並確保 AI 生成的穩定性，紙娃娃系統採用以下統一風格：

*   **核心風格**：<span style="color:#C9A227; font-weight:bold;">Pop Mart 盲盒公仔風 (Chibi 3D/Flat)</span>
*   **比例**：2.5 頭身至 3 頭身，圓潤可愛，強調頭部與眼部表情。
*   **畫風**：平塗插畫 (Flat illustration) 結合柔和賽璐璐陰影 (Soft cel-shading)。
*   **品牌色**：所有部件設計皆需融入<span style="color:#C9A227; font-weight:bold;">天命金 (#C9A227)</span> 作為點綴。

### 1.1 性別差異化設計

雖然共用同一套品牌語言，但在氣質上做出區隔：

*   **女生版**：賽博少女風 + 日系輕奢風。暖色系、柔和曲線、夢幻氛圍。
*   **男生版**：暗黑武士風 (60%) + 國風仙俠風 (40%)。冷色系（深藍、墨黑）、稜角線條、沉穩氣場。

.

## 2. 圖層架構與分類

紙娃娃系統採用「純圖層疊加 (CSS `position: absolute`)」架構，由底層至頂層共分為 7 大類：

| 圖層順序 (Z-Index) | 類別 | 包含部件 | 生成難度 | 備註 |
| :--- | :--- | :--- | :--- | :--- |
| 1 (底層) | **特效層 (後)** | 背後靈氣、五行光環 | 🟢 低 | 建議由 CSS/前端特效實現 |
| 2 | **素體層** | 身體、臉部、基礎底衣 | 🟢 低 | 固定不變，男女各一款 |
| 3 | **下半身** | 褲子、裙子 | 🟡 中 | 需與素體腰部對齊 |
| 4 | **上半身** | 上衣、外套、袍子 | 🟡 中 | 需覆蓋素體軀幹與手臂 |
| 5 | **鞋子** | 靴子、球鞋 | 🟠 中高 | 需與素體腳部精確對齊 |
| 6 | **頭部** | 髮型、帽子 | 🟡 中 | 需配合素體頭型，臉部區域需透明 |
| 7 | **配件層** | 眼鏡、手錶、手串、手持物 | 🔴 高 | 尺寸小，AI 生成細節易模糊 |
| 8 (頂層) | **特效層 (前)** | 前景光暈、飄落粒子 | 🟢 低 | 建議由 CSS/前端特效實現 |

.

## 3. 檔案規格與輸出要求

為確保系統 Agent 能順利疊加圖層，所有美術素材必須符合以下嚴格規範：

*   **畫布尺寸**：統一為 `1024 x 1024 px`。
*   **角色佔比**：角色素體高度約佔畫布的 70%，確保四周有足夠空間容納大型配件或特效。
*   **背景**：必須為<span style="color:#C9A227; font-weight:bold;">純透明背景 (Transparent PNG)</span>。
*   **對齊基準**：所有部件在生成時，必須以「素體 T-pose」為絕對參考座標。例如：生成「帽子」時，帽子在 1024x1024 畫布上的位置，必須剛好落在素體頭部的位置。

.

## 4. 可行性評估與開發建議

經過實際生成男女兩套完整範例（詳見 `ART/OUTPUTS/TASK-003/` 目錄），得出以下評估結論：

### 4.1 成功驗證項目

1.  **風格統一性**：AI 能穩定產出符合 Pop Mart 盲盒風格的部件，且男女版本風格一致。
2.  **大部件對齊**：上衣、下著、髮型等大面積部件，能與素體良好對齊。

### 4.2 潛在風險與解法

1.  **微小配件細節流失**：在 Chibi 比例下，手錶、手串等微小配件的細節會被壓縮。
    *   *解法*：配件設計需「圖示化、誇張化」，避免過度寫實的細節。
2.  **邊緣像素溢出**：AI 生成透明背景時，偶爾會有極少量的像素溢出或邊緣不乾淨。
    *   *解法*：美術 Agent 需在交付前進行人工檢查與去背優化。
3.  **尺寸微調需求**：部分部件（如鞋子、帽子）生成後，可能需要微調比例才能完美貼合素體。
    *   *解法*：建議系統 Agent 在前端實作時，保留對個別圖層進行 `transform: scale()` 微調的彈性。

.
