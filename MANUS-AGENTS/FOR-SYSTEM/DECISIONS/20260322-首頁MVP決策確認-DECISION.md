## 首頁架構決策

### 路由調整
- 現有首頁（登入頁）→ 移至 `/login`
- 新首頁 `/` → 成為主要入口（公開可見，無需登入）
- **已登入用戶**：系統偵測到 session cookie → 自動跳轉至功能頁（舊首頁位置）
- **未登入用戶**：看到新首頁，可體驗「錦囊體驗」

### 三大區塊
1. **錦囊體驗**（核心）：輸入生日 → 換算命格 → 顯示體驗版演示
2. **功能卡片**：命格身份證預覽、命理日曆、專家直通
3. **社會認證**：用戶見證文字

## 錦囊體驗 API 設計決策

### 設計原則
- 給**未登入新用戶**使用，不需要 Token
- 輸入生日後，後端自動換算命格（八字、生命靈數、農曆）
- 只回傳**體驗版演示資料**，不開放完整命盤

### 開放功能範圍（體驗版）
| 功能 | 體驗版內容 | 完整版差異 |
|------|-----------|-----------|
| **每日運勢** | 今日整體運勢摘要（1-2 句）+ 幸運色 | 完整五行分析、時辰建議 |
| **命格解析** | 命格類型名稱 + 一句話描述 | 完整日主五行、喜用神、格局 |
| **天命問卜** | 單一問題示範回應 | 完整 AI 問卜對話 |

### API 端點設計
- 新建公開端點：`GET /api/preview?birth=YYYY-MM-DD`
- 不需要 Token，任何人可呼叫
- 回傳精簡版命格資料（控制資訊量，引導轉換）

### 轉化路徑
```
訪客輸入生日 → 看到體驗版演示 → 顯示「登入後解鎖完整命盤」CTA → 引導至 /login
```
