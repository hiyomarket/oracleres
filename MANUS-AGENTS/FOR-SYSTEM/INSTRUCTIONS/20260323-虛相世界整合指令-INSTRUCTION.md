## 一、最重要的事：大廳要改

請先閱讀這段，再看其他技術細節。

**現況**：大廳（GAME LOBBY）是三張並列卡片（靈相世界 / 靈相空間 / 天命商城）。

## 五、命格資料串接（最關鍵的技術點）

虛相世界旅人的初始能力值，必須從你那邊的 `natalStats` 讀取。

**你需要提供的 API**（或 tRPC endpoint）：

```typescript
// 呼叫方式（我這邊會這樣呼叫）
const natalStats = await fetchNatalStats(userId);

// 預期回傳格式
{
  userId: string,
  gender: "male" | "female",
  natalStats: {
    wood: number,   // 0-100，百分比
    fire: number,
    earth: number,
    metal: number,
    water: number
  }
}
```

**換算邏輯（我這邊已實作）**：
- maxHp = wood% × 5 + 50
- attack = fire% × 0.8 + 5
- defense = earth% × 0.6 + 5
- speed = metal% × 0.4 + 3
- maxMp = water% × 3 + 20

## 六、任務祝福加成串接

當玩家完成任務（例如今日穿搭任務），需要通知虛相世界給旅人加成。

**呼叫方式**（你那邊完成任務後呼叫我這邊）：

```typescript
// POST /api/game/blessing
{
  userId: string,
  blessingType: "hp_boost" | "attack_boost" | "exp_boost" | "gold_boost",
  value: number,      // 加成數值或百分比
  duration: number,   // 持續 Tick 數（-1 = 永久）
  source: string      // 來源描述，例如 "今日穿搭任務完成"
}
```

這個介面我這邊已預留，等你確認呼叫方式後立即實作。
