## P0 — 立即修正（影響核心戰鬥邏輯）

### 修正一：魔法傷害精神比例公式

**問題**：GD-005 把魔法係數 A 的「精神比例」固定為「自身水屬性 / 敵方水屬性」，導致非水命玩家的魔法傷害係數長期停在 0.1（最低檔），魔法技能形同廢棄。

**修正方式**：將精神比例改為「施法者對應技能屬性 / 敵方對應抗性」。

具體而言，每個技能都有一個「主屬性標籤」（例如炎爆術 = 火屬性技能），計算時使用施法者的火屬性值與敵方的爆炸/範圍抗性相比：

```typescript
// 修正前（錯誤）
const spiritRatio = attacker.water / defender.water * 100;

// 修正後（正確）
const skillElement = skill.element; // 技能的五行屬性標籤
const attackerElementValue = attacker[skillElement]; // 施法者對應屬性值
const defenderResistance = getDefenderResistance(defender, skillElement); // 敵方對應抗性
const spiritRatio = (attackerElementValue / defenderResistance) * 100;
```

五行屬性 → 敵方抗性的對應關係（依 GD-005 第十二章）：

| 技能五行 | 對應敵方抗性類型 |
|:---:|:---|
| 木 | 毒/腐蝕抗性 |
| 火 | 爆炸/範圍抗性 |
| 土 | 物理抗性 |
| 金 | 精神抗性 |
| 水 | 魔法抗性 |

### 補充二：命盤五行比例 → 屬性數值統一換算公式

**問題**：目前 `getEquipped` 回傳的 `natalStats`（木/火/土/金/水百分比）與角色面板的 HP/ATK/DEF/SPD/MATK/MP 之間缺乏統一的換算公式，各處實作可能不一致。

**請統一使用以下公式**（詳見 GD-020 補充二）：

```typescript
function calcCharacterStats(natalStats: NatalStats, level: number) {
  const { wood, fire, earth, metal, water } = natalStats;
  return {
    hp:   Math.floor(100 + level * 10 + wood  * 3.0),
    atk:  Math.floor(10  + level * 2  + fire  * 1.5),
    def:  Math.floor(10  + level * 1.5 + earth * 1.5),
    spd:  Math.floor(5   + level * 0.5 + metal * 0.8),
    matk: Math.floor(10  + level * 2   + water * 1.5),
    mp:   Math.floor(50  + level * 5   + water * 1.5),
  };
}
```

> `natalStats` 的值為 0~100 的整數（百分比），例如木 42 代表木屬性 42%。

**請確認以下位置都使用此公式**：
- `server/routers.ts` 或 `server/routers/game.ts` 中建立旅人的邏輯
- `client/src/pages/CharacterProfile.tsx` 或角色面板的顯示邏輯
- 虛相世界沙盒的 `server/tickEngine.ts`（戰鬥計算部分）

### 補充三：戰鬥日誌呈現規格

放置遊戲的戰鬥日誌是玩家唯一的戰鬥體驗媒介，需要有統一的文字格式。

**每回合戰鬥日誌格式**：

```
[旅人名稱] 對 [怪物名稱] 發動 [技能/普攻]！
→ [五行加成說明（如有）]
→ 造成 [傷害數字] 點傷害[（天命一击！）]
→ [怪物名稱] 剩餘 HP：[當前HP] / [最大HP]
```

**不應顯示**：係數 A/B/C 數值、240 閾值計算過程、傷害波動倍率。

完整日誌格式規範請見 GD-020 補充三。

### 補充四：怪物種族欄位補充

現有 `shared/monsters.ts` 的 50 隻怪物缺少 `race`（種族）欄位，導致 GD-005 的種族剋制矩陣無法運作。

**請在怪物資料結構中新增 `race` 欄位**：

```typescript
type MonsterRace = 
  | 'spirit_beast'  // 靈獸系
  | 'undead'        // 亡魂系
  | 'metal'         // 金屬系
  | 'humanoid'      // 人型系
  | 'plant'         // 植物系
  | 'aquatic'       // 水生系
  | 'demon'         // 妖化系
  | 'dragon'        // 龍種系
  | 'insect'        // 蟲類系
  | 'destiny';      // 天命系（Boss 專屬）
```

各怪物的種族對應請見 GD-020 補充四的「台灣地圖怪物種族對應表」。
