在 `drizzle/schema.ts` 中新增以下六張表（完整 TypeScript 定義在 GD-022 第二章）：

| 表名 | 用途 |
|------|------|
| `skill_templates` | 技能靜態資料（350 種技能的完整屬性） |
| `agent_skills` | 角色已習得技能（含覺醒階段、使用次數、裝備槽位） |
| `skill_books` | 背包中的技能書道具 |
| `awake_materials` | 覺醒素材庫存（屬性精華 × 5 種 + 天命碎片） |
| `hidden_skill_trackers` | 隱藏技能觸發條件追蹤計數 |
| `global_first_triggers` | 全服首觸發記錄 |

執行：
```bash
pnpm db:push
```

### 任務 4：在 Tick 引擎加入 Combo 計算

在 `server/tickEngine.ts` 的戰鬥結算流程中加入 Combo 計算（完整程式碼在 GD-022 第四章）：

1. 在每次戰鬥 Tick 開始時初始化 `ComboBuildState`
2. 按照角色技能槽位的優先序依次使用技能
3. 每次使用技能後呼叫 `calculateComboBonus()` 計算加成
4. 將 Combo 加成結果寫入事件日誌（例：「木火協力！燃燒傷害 +50%」）
