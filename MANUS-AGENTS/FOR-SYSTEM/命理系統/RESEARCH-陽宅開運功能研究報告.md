## 一、陽宅開運是什麼

用戶輸入自宅格局（門向、床位、爐灶方位等），系統根據其命卦五行喜忌，給出個人化風水開運建議。

**定位**：將傳統命理諮詢數位化，與用戶命格綁定，差異化競爭力強。

### 4.2 系統二：八宅理論（陽宅風水核心）

**命卦計算公式（命理顧問提供）：**
```
男性：（100 - 出生年末兩位）÷ 9 取餘數 + 1 → 對應卦位
女性：（9 - 出生年末兩位）÷ 9 取餘數 + 1 → 對應卦位

卦位對照：1坎/2坤/3震/4巽/5坤/6乾/7兑/8艮/9離
```

**東四命 vs 西四命：**

| 命卦 | 分類 | 宜居方位 |
|:---|:---:|:---|
| 坎、離、震、巽 | 東四命 | 東、南、東南、北 |
| 坤、兌、乾、艮 | 西四命 | 西、西北、西南、東北 |

**八宅核心原則：**
- 門、主、灶 三者宮位關係需配合
- 游星飛泊（貪狼、巨門、武曲等）影響每年方位

### 4.3 飛星盤邏輯（流年動態）

```
根據入住年份計算九宮飛星分布
每年飛星不同 → 當前旺位也不同
流年逢財庫（辰丑）→ 財位加強
流年逢官非（午未）→ 小人位化解
```

## 九、上班族陽宅商業化附錄（命理顧問產出）

### 9.1 切入點：「優化自己的工位」

> 與其叫老闆換風水，不如先優化自己的工位。
> 這個切入點對上班族心理門檻低，容易傳播。

### 9.2 辦公桌風水核心問題（6個）

| 問題 | 對應改善目標 |
|:---|:---|
| 你的座位方向 | 對應命卦財位/官位 |
| 背後是否有實牆 | 貴人靠山 |
| 座位是否對門/對窗 | 小人/漏財 |
| 電腦螢幕方向 | 事業/學業 |
| 桌上左手邊是否有筆筒/文件 | 龍動虎靜 |
| 抽屜是否凌亂 | 財庫意象 |

### 9.3 上班族陽宅商業變現層級

| 層級 | 方案 | 價格 | 說明 |
|:---|:---|---:|:---|
| 免費鉤子 | 今日工位提醒 | 免費 | 根據流日干支推薦座位方向 |
| 低單價付費 | 個人辦公桌分析報告 | 49元 | 5分鐘產出 |
| 低單價付費 | 月度工位調整建議 | 99元/月 | 結合流月 |
| 高單價付費 | 完整職場陽宅分析 | 399元 | 座位、會議室選擇、升遷財位 |
| O2O | 線上視訊陽宅師 | 1500–3000元/次 | 針對店面/公司/辦公室現場勘查 |
| 訂閱制 | 每週職場能量提醒 | 299元/月 | 工位方向、流日吉凶、本週小人/貴人方位 |

### 9.4 差異化核心（別人做不到的）

| 功能 | 說明 |
|:---|:---|
| 命卦 + 行業五行 | 餐飲業、金融業、科技業的「旺位方向」不同 |
| 流年 + 職場階段 | 新人求穩，中層求升，高層求權 |
| 座位時間軸 | 「這禮拜坐這裡，下禮拜要換」動態建議 |
| 小人化解衛星空缺 | 特定時段坐在哪個位置避開小人 |

### 9.5 命卦計算公式（數字轉卦位）

```
男性：（100 - 出生年末兩位）÷ 9 取餘數 + 1
女性：（9 - 出生年末兩位）÷ 9 取餘數 + 1

餘數對照：
1 → 坎（北）
2 → 坤（西南）
3 → 震（東）
4 → 巽（東南）
5 → 坤/寄宮（中）
6 → 乾（西北）
7 → 兌（西）
8 → 艮（東北）
9 → 離（南）
```

### 9.6 八宅 × 流年飛星聯動邏輯

```
流年飛星：每年九宮飛星不同
↓
查表：該年哪個宮位有財星/官星/小人星
↓
對照用戶命卦（東四命/西四命）
↓
輸出：當前最旺方位 + 當前需避開方位
↓
結合用戶輸入的座位方向
↓
給出「這週座位建議」與「下週可能要換的方向」
```

## 十一、AI辦公室建模：可行方案（2026-04-02 新增）

### 方案比較

| 方案 | 做法 | 難度 | 適合階段 |
|:---|:---|:---:|:---|
| A. 純表單+陀螺儀 | 用戶回答6問題+手機羅盤校準方向 | ⭐ 低 | MVP首選 |
| B. 照片+AI解析 | 上傳照片，AI標註物體位置，用戶確認 | ⭐⭐ 中 | 第二版 |
| C. AR即時疊加 | 手機鏡頭對準，AR顯示財位/貴人位標記 | ⭐⭐⭐⭐ 高 | O2O演示 |

### 照片能提供什麼（命理顧問定義）

| 輸入項目 | 照片能否提供 | 需要的技術 |
|:---|:---:|:---|
| 座位方向 | ❌ 無法 | 需要羅盤功能或手機陀螺儀+GPS |
| 背後是否有實牆 | ✅ 可以 | 照片AI分析 |
| 是否對門/對窗 | ✅ 可以 | 照片AI分析 |
| 螢幕方向 | ⚠️ 模糊 | 需要用戶標註或AI推斷 |
| 左手邊擺設 | ✅ 可以 | 照片AI分析 |
| 抽屜是否凌亂 | ⚠️ 有限 | 開放式抽屜可分析 |

### MVP技術路徑

```
用戶上傳一張辦公桌全景照片
    ↓
AI視覺分析（GPT-4V）：
  - 背後是否有牆
  - 是否對門/對窗
  - 左手邊有什麼
    ↓
用戶搖手機校準一次方向（陀螺儀）
    ↓
命理引擎輸出：
  - 財位方向
  - 貴人位
  - 建議擺放清單
```

**技術需求：照片上傳（已有）+ AI視覺解析（GPT-4V API）+ 陀螺儀方向（HTML5 API）+ 命理引擎（wuxing.ts已有）**
