## 任務目標摘要
為天命共振網站（oracleres.com）建置全新首頁 MVP：

### 三大區塊
1. **錦囊體驗** — 輸入生日 → 換算命格 → 體驗版演示（每日運勢摘要 + 命格解析 + 天命問卜示範）

3. **社會認證** — 用戶見證文字

### 現有架構變更
- 現有首頁（登入頁）→ 移至 `/login`
- 新首頁 → 成為主要入口
- **已登入用戶** → 自動跳轉功能頁（舊有登入記錄保留）
- **未登入用戶** → 看到新首頁，可體驗錦囊功能

### API 設計
- 新建公開端點：`GET /api/preview?birth=YYYY-MM-DD`
- 不需要 Token，回傳精簡版命格資料
