## 一、設計核心理念

角色的初始屬性由用戶的真實命盤五行比例轉化而來，命格決定起點，但不設職業限制。玩家可以透過在大地圖上收集五行能量，慢慢補足弱項，打造任意流派。

---

## 二、行動力數值（雙數值系統）

這兩個數值控制玩家的「行動頻率」，是免費/付費差異的核心。

| 數值 | 定義 | 消耗方式 | 自然恢復 | 上限 |
| :--- | :--- | :--- | :--- | :--- |
| **體力值 (Stamina)** | 用戶親自操作時消耗 | 手動打卡、主動採集、戰鬥 | 每 30 分鐘 +1 | 10 點 |
| **活躍值 (Activity)** | 角色自動派遣任務時消耗 | 掛機自動任務（用戶不在線也能跑） | 每 2 小時 +1 | 5 點 |

**付費加速選項**：
- 購買「靈氣補充瓶」→ 體力值立即回滿
- 購買「天命令牌」→ 活躍值立即回滿
- 訂閱方案 → 恢復速度提升（體力每 15 分鐘 +1，活躍每 1 小時 +1）

## 三、五行屬性：三維轉化系統

用戶命盤的五行比例轉化為角色的三個維度屬性，每個五行在三個維度上都有獨立的定義。

### 3.1 完整對應表

| 五行 | 角色素質 | 生活系 | 戰鬥系 |
| :--- | :--- | :--- | :--- |
| **木** | 智力（技能學習速度、技能槽數量） | 採集力（植物/草藥類掉落率 +） | 治癒力（戰鬥中自我回血量） |
| **火** | 力量（可裝備武器的重量上限） | 鍛冶力（製造武器/裝備的成功率與品質） | 攻擊力（物理傷害基礎值、暴擊率） |
| **土** | 體魄（HP 上限與自然回血速度） | 承重力（背包格子數量、可攜帶道具總重） | 防禦力（物理傷害減免） |
| **金** | 敏捷（行動速度、閃避率） | 精煉力（提升素材品質等級的機率） | 命中力（攻擊命中率、穿透敵方防禦比例） |
| **水** | 精神/魔力（魔法技能施放次數與強度） | 尋寶力（地圖可視範圍、稀有素材/寵物出現率） | 魔法攻擊（元素傷害基礎值、狀態異常觸發率） |

### 3.2 命盤轉化範例

以 Boss（蘇祐震先生）的命盤為例：
**木 42%, 水 35%, 火 11%, 土 9%, 金 4%**

| 屬性 | 初始值（滿分 100） | 角色特徵 |
| :--- | :--- | :--- |
| 木（智力/採集/治癒） | 42 | 技能槽多、採集強、戰鬥中回血能力佳 |
| 水（精神/尋寶/魔攻） | 35 | 魔攻強、地圖可視範圍廣、稀有物品發現率高 |
| 火（力量/鍛冶/攻擊） | 11 | 物理攻擊弱、鍛造能力普通 |
| 土（體魄/承重/防禦） | 9 | HP 少、防禦低、背包格子少 |
| 金（敏捷/精煉/命中） | 4 | 移動慢、命中率低、精煉能力弱 |

> **初始角色定位**：高智力 + 高精神力 → 天生的「術士/法師型」，技能槽多、魔攻強、探索廣，但物理戰鬥能力弱，需要靠補足火土金能量來強化弱項。

## 四、抗性系統

抗性是防禦的進階層次，對應不同類型的傷害來源。

| 抗性類型 | 對應五行 | 說明 | 提升方式 |
| :--- | :--- | :--- | :--- |
| **物理抗性** | 土（體魄） | 減少物理攻擊傷害 | 土屬性提升、土系裝備 |
| **魔法抗性** | 水（精神） | 減少元素/魔法傷害 | 水屬性提升、水系護符 |
| **毒/腐蝕抗性** | 木（生命力） | 減少持續性傷害（中毒、詛咒） | 木屬性提升、解毒類道具 |
| **精神抗性** | 金（意志） | 抵抗控制技能（暈眩、恐懼、魅惑） | 金屬性提升、金系護符 |
| **爆炸/範圍抗性** | 火（爆發） | 減少爆炸性/範圍傷害 | 火屬性提升、防爆類裝備 |

**抗性護符系統**：
玩家在地圖上採集特定素材，可以在自宅的「煉製台」製作對應的護符：
- 採集「寒冰水晶」→ 煉製「冰霜護符」→ 魔法抗性 +15%
- 採集「玄鐵礦石」→ 煉製「玄鐵護甲」→ 物理抗性 +20%

## 五、五行相生相剋戰鬥機制

戰鬥中的屬性剋制依照五行相剋關係設計，讓低等級玩家也能透過「帶對屬性」挑戰高等怪物：

| 剋制關係 | 效果 |
| :--- | :--- |
| 木剋土 | 木屬性攻擊對土屬性怪物傷害 +25% |
| 土剋水 | 土屬性防禦對水屬性攻擊減免 +25% |
| 水剋火 | 水屬性魔攻對火屬性怪物傷害 +25% |
| 火剋金 | 火屬性攻擊對金屬性怪物傷害 +25% |
| 金剋木 | 金屬性命中對木屬性怪物穿透 +25% |

## 六、地圖可視範圍系統

可視範圍是「水屬性（精神/尋寶力）」最重要的體現，決定玩家在大地圖上能觀察到的內容範圍。

| 可視範圍等級 | 水屬性門檻 | 能看到的內容 |
| :--- | :--- | :--- |
| 基礎視野（1格） | 0-20 | 普通採集點、基礎怪物、一般商店 |
| 延伸視野（2格） | 21-40 | 稀有採集點、精英怪物、隱藏寶箱 |
| 廣域視野（3格） | 41-60 | 流浪商人位置、技能導師位置、寵物棲息地 |
| 天眼視野（4格） | 61-80 | 隱藏地圖入口、Boss 刷新點、其他玩家活動軌跡 |
| 命格天眼（5格） | 81+ | 天命節點、傳說級素材、隱藏劇情觸發點 |

**補足可視範圍的三種方法**：
1. 提升水屬性（長期養成）
2. 購買「千里眼符」道具（短期消費）
3. 攜帶水屬性的隨行型寵物（被動加成）

## 七、角色成長路徑

1. **初始**：命盤五行比例直接轉化為初始屬性
2. **日常成長**：在地圖上採集五行能量，注入自宅「修煉室」提升對應屬性
3. **裝備強化**：穿戴裝備獲得屬性加成
4. **技能解鎖**：達到天賦條件後自動解鎖隱藏技能
5. **寵物輔助**：攜帶對應屬性的寵物補足弱項
