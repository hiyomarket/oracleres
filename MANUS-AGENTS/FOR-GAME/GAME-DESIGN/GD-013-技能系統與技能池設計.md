## 一、設計核心理念

技能收集是《靈相虛界》長線遊玩的核心動力之一。為了滿足不同類型玩家的需求，技能池不僅包含「戰鬥技能」，還涵蓋了「生活採集」、「鍛造精煉」與「探索輔助」等多元領域。

技能分為五個階級：**初階、中階、高階、傳說、天命**。玩家可以透過商店購買、怪物掉落、流浪商人、隱藏任務或命理事件來收集這些技能。技能之間存在互相牽引與平衡的關係，例如強大的戰鬥技能可能需要消耗特定的生活素材，而高級的生活技能則需要透過戰鬥擊敗特定 Boss 來解鎖。

## 二、技能階級與獲取途徑

| 階級 | 顏色 | 獲取途徑 | 說明 |
| :--- | :--- | :--- | :--- |
| **初階** | 白色 | 基礎商店購買、新手任務 | 基礎實用技能，無特殊消耗 |
| **中階** | 綠色 | 精英怪掉落、進階商店 | 具備一定戰術價值，可能需要消耗 MP |
| **高階** | 藍色 | Boss 掉落、流浪商人 | 強力技能，通常帶有特殊效果或屬性加成 |
| **傳說** | 金色 | 隱藏商人、極低機率掉落 | 改變遊戲玩法的核心技能，極難獲取 |
| **天命** | 紫色 | 命理事件觸發、天命幣商城 | 與玩家本命五行深度綁定，具備唯一性 |

## 六、技能系統的互相牽引與平衡

1. **資源循環**：強大的戰鬥技能（如 SK_C011 萬木復甦）每次使用可能需要消耗特定的生活素材（如靈泉水），促使戰鬥玩家必須參與生活採集，或向生活玩家購買。
2. **解鎖門檻**：高級的生活技能（如 SK_L008 靈獸親和）需要擊敗強大的 Boss 才能獲得，這鼓勵生活玩家提升一定的戰鬥力，或尋求戰鬥玩家的組隊幫助。
3. **命理限制**：天命級技能雖然強大，但往往受到玩家「流日」或「本命五行」的限制，無法無腦濫用，確保了遊戲的平衡性與策略深度。
