## 一、設計核心理念

先提出我的看法：目前的戰鬥系統（打怪、捕捉）偏向硬核玩家，但天命共振的受眾有很大一部分是「生活性玩家」。他們不一定喜歡計算傷害倍率，但他們會為了「在地圖上撿到一顆發光的石頭」或「遇到一個賣稀有家具的流浪商人」而每天打開 App。

再來執行你的提案：本系統旨在擴充 GD-003（大地圖系統），加入類似《皮克敏 (Pikmin Bloom)》的輕度生活互動機制。透過「採集點」、「隨機任務」與「流浪商人」，讓玩家在現實生活中的通勤、散步，都能轉化為遊戲內的驚喜與回饋。

## 二、採集系統 (Gathering)

地圖上除了怪物，還會隨機生成「採集點」。採集不需要戰鬥，只需點擊即可獲得資源。

### 2.1 採集點類型與五行對應

| 採集點外觀 | 對應五行 | 產出資源 | 現實生成邏輯 (LBS) |
| :--- | :--- | :--- | :--- |
| **發光的靈草** | 木 | 藥草（用於合成恢復道具） | 公園、綠地、學校附近機率高 |
| **溫暖的餘燼** | 火 | 火晶石（用於武器鍛造） | 餐廳、廟宇、白天陽光下機率高 |
| **奇特的礦石** | 土 | 礦石（用於防具鍛造） | 山區、建築工地附近機率高 |
| **閃爍的金屬** | 金 | 零件（用於家具製作） | 商業區、車站、百貨公司機率高 |
| **純淨的水滴** | 水 | 靈泉（用於寵物培養） | 河岸、海邊、雨天機率大幅提升 |

### 2.2 採集機制
- **體力消耗**：每次採集消耗 1 點體力（上限 100，每 10 分鐘恢復 1 點）。
- **屬性加成**：玩家的「木屬性（智力/採集）」越高，單次採集獲得的數量越多，且有更高機率獲得「稀有品質」的素材。

## 五、與現有系統的整合

1. **與自宅系統 (GD-007)**：採集到的素材可用於製作家具，佈置起居室。
2. **與商城系統 (GD-008)**：流浪商人作為商城的補充，提供「肝帝（重度遊玩者）」獲取付費道具的另一種可能。
3. **與 Game CMS**：所有的採集物、隨機任務文本、流浪商人商品清單，都必須能透過 Game CMS 後台手動配置。
