# GD-009：大地圖生活互動系統設計文檔 (皮克敏式玩法)

**版本**：v1.0
**最後更新**：2026-03-23
**負責人**：遊戲 Agent
**狀態**：`已確認`

---

## 一、設計核心理念

先提出我的看法：目前的戰鬥系統（打怪、捕捉）偏向硬核玩家，但天命共振的受眾有很大一部分是「生活性玩家」。他們不一定喜歡計算傷害倍率，但他們會為了「在地圖上撿到一顆發光的石頭」或「遇到一個賣稀有家具的流浪商人」而每天打開 App。

再來執行你的提案：本系統旨在擴充 GD-003（大地圖系統），加入類似《皮克敏 (Pikmin Bloom)》的輕度生活互動機制。透過「採集點」、「隨機任務」與「流浪商人」，讓玩家在現實生活中的通勤、散步，都能轉化為遊戲內的驚喜與回饋。

## 二、採集系統 (Gathering)

地圖上除了怪物，還會隨機生成「採集點」。採集不需要戰鬥，只需點擊即可獲得資源。

### 2.1 採集點類型與五行對應

| 採集點外觀 | 對應五行 | 產出資源 | 現實生成邏輯 (LBS) |
| :--- | :--- | :--- | :--- |
| **發光的靈草** | 木 | 藥草（用於合成恢復道具） | 公園、綠地、學校附近機率高 |
| **溫暖的餘燼** | 火 | 火晶石（用於武器鍛造） | 餐廳、廟宇、白天陽光下機率高 |
| **奇特的礦石** | 土 | 礦石（用於防具鍛造） | 山區、建築工地附近機率高 |
| **閃爍的金屬** | 金 | 零件（用於家具製作） | 商業區、車站、百貨公司機率高 |
| **純淨的水滴** | 水 | 靈泉（用於寵物培養） | 河岸、海邊、雨天機率大幅提升 |

### 2.2 採集機制
- **體力消耗**：每次採集消耗 1 點體力（上限 100，每 10 分鐘恢復 1 點）。
- **屬性加成**：玩家的「木屬性（智力/採集）」越高，單次採集獲得的數量越多，且有更高機率獲得「稀有品質」的素材。

## 三、隨機任務系統 (Random Encounters)

在地圖上走動時，除了固定節點，玩家的「可視範圍」內會隨機刷出 NPC 或事件氣泡。

### 3.1 任務類型

| 任務類型 | 觸發情境 | 任務內容範例 | 獎勵 |
| :--- | :--- | :--- | :--- |
| **尋物委託** | 遇到迷路的靈相居民 | 「我弄丟了 3 顆火晶石，你能幫我找回來嗎？」 | 靈石、Aura Score |
| **五行調和** | 遇到失衡的地脈節點 | 「此地水氣過盛，請提交 5 個土屬性素材來鎮壓。」 | 大量對應五行能量 |
| **限時快遞** | 遇到旅行商人 | 「請在 30 分鐘內走到下一個地圖節點交貨。」 | 稀有家具圖紙 |

### 3.2 命理連動（高衝擊目標）
隨機任務的生成會與玩家的「今日流日運勢」連動。例如：
- 若今日流日為「破財」，地圖上會更容易遇到「需要你捐獻物資」的乞討 NPC，完成後可獲得「破財消災」的隱藏 Buff（提升明日的掉寶率）。
- 若今日流日為「桃花」，地圖上會更容易遇到「需要幫忙傳遞情書」的任務，獎勵為提升寵物親密度的道具。

## 四、流浪商人系統 (Wandering Merchants)

流浪商人是地圖上最稀有、最讓玩家期待的隨機事件。

### 4.1 出現機制
- **機率**：極低（每日約 5% 機率在玩家附近生成）。
- **停留時間**：生成後僅停留 2 小時，超時即消失。
- **屬性加成**：玩家的「水屬性（尋寶力）」越高，越容易在地圖上看到流浪商人的圖示。

### 4.2 販售商品
流浪商人**不收天命幣**，只收「靈石」或「特定稀有素材」。這確保了免費用戶也能透過努力獲得頂級物品。

| 商品類型 | 說明 | 價格區間 |
| :--- | :--- | :--- |
| **絕版家具** | 只能向流浪商人購買的自宅裝飾品 | 500 - 2000 靈石 |
| **神秘盲盒** | 打開可能獲得天命級裝備或一堆垃圾 | 300 靈石 |
| **高級捕捉符** | 商城賣 100 天命幣，這裡可用靈石買 | 1000 靈石 |
| **命理秘卷** | 使用後直接解鎖一次「深度命盤解析」 | 需用 50 個稀有素材兌換 |

## 五、與現有系統的整合

1. **與自宅系統 (GD-007)**：採集到的素材可用於製作家具，佈置起居室。
2. **與商城系統 (GD-008)**：流浪商人作為商城的補充，提供「肝帝（重度遊玩者）」獲取付費道具的另一種可能。
3. **與 Game CMS**：所有的採集物、隨機任務文本、流浪商人商品清單，都必須能透過 Game CMS 後台手動配置。
