## 一、設計核心理念

本系統建立在「靈相/虛相雙世界架構」之上。在虛相世界（RPG 冒險）中，裝備與道具的呈現以極簡的「像素風格」與「純文字列表」為主，大幅降低美術成本，從而允許我們設計極度豐富的裝備部位與道具種類，支撐起龐大的遊戲經濟與戰鬥策略。

## 二、裝備系統設計

### 2.1 裝備品質分級

裝備品質分為五個等級，對應五行色彩，品質越高，附加屬性與特殊效果越強：

| 品質等級 | 顏色標示 | 獲取難度 | 說明 |
| :--- | :--- | :--- | :--- |
| **普通** | 白色 | 極易 | 基礎商店購買、低級怪物掉落 |
| **精良** | 綠色 | 容易 | 玩家鍛造、一般怪物掉落 |
| **稀有** | 藍色 | 中等 | 精英怪掉落、高級鍛造 |
| **傳說** | 金色 | 困難 | Boss 掉落、特殊配方鍛造 |
| **天命** | 紅色 | 極難 | 命理特殊活動限定、隱藏任務 |

### 2.2 裝備部位與屬性加成

虛相世界的像素小人支援多部位換裝，每個部位提供不同的核心加成：

| 裝備部位 | 主要屬性加成 | 說明 |
| :--- | :--- | :--- |
| **武器** | 攻擊力、魔力 | 決定攻擊方式與五行屬性（詳見 2.3） |
| **頭盔** | 防禦、精神抗性 | 保護頭部，減少異常狀態機率 |
| **上衣** | 防禦、HP 上限 | 主要防禦裝備 |
| **下衣** | 速度、承重 | 影響行動力與背包容量 |
| **鞋子** | 速度、閃避 | 影響戰鬥先手與逃跑成功率 |
| **戒指（左/右）** | 各屬性微調 | 兩個獨立欄位，提供靈活的數值搭配 |
| **項鍊** | 特殊效果 | 如暴擊率提升、特定五行抗性 |
| **護符** | 命理加成 | 天命共振獨有，如「天命吉日全屬性 +5%」 |

### 2.3 武器系統（無職業限制設計）

武器不綁定職業，而是提供不同的屬性加成，由玩家搭配技能來決定戰鬥流派。

| 武器類型 | 主要加成 | 次要加成 | 適合搭配流派 |
| :--- | :--- | :--- | :--- |
| **單手劍** | 攻擊力 + | 命中率 + | 物理攻擊技能、防禦技能（另一手持盾） |
| **雙手劍** | 攻擊力 ++ | 速度 - | 高倍率物理技能，捨棄副手欄 |
| **弓** | 攻擊力 +、命中率 ++ | 速度 + | 遠程物理技能、暴擊流 |
| **槍/長槍** | 攻擊力 +、速度 + | — | 先手流、連擊技能 |
| **法杖（標準）** | 魔力 ++ | 精神 + | 魔法攻擊技能、元素技能 |
| **力量魔法杖** | 魔力 +、力量 + | — | 物魔混合流（天堂系設計） |
| **短刃/匕首** | 速度 ++、暴擊率 + | 攻擊力 - | 刺客流、毒技能、先手必殺 |
| **拳套** | 攻擊力 +、速度 + | 防禦 + | 近戰連擊技能、格鬥流 |
| **聖器（護符類）** | 精神 +、魔力 + | 五行抗性 + | 輔助/治癒技能、支援型打法 |

*註：每把武器皆有自身的五行屬性，攻擊時可觸發五行相剋加成。*

## 三、道具系統設計

在虛相世界中，道具以純文字列表呈現（如：`神行符 ×2`），使用時僅顯示文字提示，無須複雜動畫。

### 3.1 道具分類

| 道具類別 | 範例 | 說明 |
| :--- | :--- | :--- |
| **恢復類** | 小金創藥、還魂丹 | 戰鬥中或地圖上恢復 HP/MP |
| **增益類（戰鬥）** | 狂暴符、鐵甲符 | 戰鬥中短暫提升屬性（限單場） |
| **探索類（地圖）** | 神行符、千里眼符 | 增加移動速度、擴大可視範圍 |
| **捕捉類** | 靈獸誘餌、高級捕捉符 | 用於戰鬥中捕捉魔物 |
| **鑑定與養成類** | 潛力鑑定卷軸、五行精華 | 鑑定魔物個體值、提升寵物經驗 |
| **素材類** | 鐵礦石、百年靈芝 | 用於鍛造裝備或製作其他道具 |

### 3.2 道具獲取與消耗循環

- **獲取**：地圖採集、戰鬥掉落、商城購買（靈相世界購買後存入虛相倉庫）。
- **消耗**：戰鬥使用、鍛造消耗、提交任務。
- **經濟意義**：道具是玩家消耗多餘素材的主要出口，也是天命幣在遊戲內的核心消費場景。
