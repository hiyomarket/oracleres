```
┌─────────────────────────────────────────────────────────────┐
│  掉落率遞減公式：                                           │
│                                                             │
```

### 4.2 五行屬性對應掉落加乘

```
┌─────────────────────────────────────────────────────────────┐
│  屬性相剋掉落加成：                                           │
│                                                             │
│  使用被剋制屬性攻擊 → 掉落率 +50%                            │
│  使用無關屬性攻擊 → 掉落率 +0%                              │
│  使用相生屬性攻擊 → 掉落率 -20%                             │
│                                                             │
│  例：金屬性武器打木系怪 → 掉落率 ×1.5                        │

└─────────────────────────────────────────────────────────────┘
```

### 4.3 掉落率計算公式

```
最終掉落率 = 基礎掉落率 × 屬性加成 × 幸運加成 × 任務加成

其中：
- 基礎掉落率：怪物種類決定
- 屬性加成：使用被剋制屬性時 ×1.5
- 幸運加成：玩家幸運值 / 1000（最高 +50%）
- 任務加成：特定任務期間 ×1.2
```
