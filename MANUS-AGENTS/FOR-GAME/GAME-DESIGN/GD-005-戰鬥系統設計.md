## 一、設計核心理念

戰鬥系統採用經典的「橫式回合制 RPG」架構（類似 FF6、魔力寶貝）。此設計能大幅降低美術與開發成本，同時保留足夠的策略深度，讓五行相剋與技能搭配的樂趣得以完整發揮。

---

## 二、戰鬥介面設計

### 2.1 畫面佈局

戰鬥畫面分為左右兩側，下方為資訊與操作區：

```text
┌─────────────────────────────────────────┐
│  [敵方區域 - 左側]    [玩家區域 - 右側]  │
│                                         │
│  怪物立繪 × 1-4       角色立繪 × 1-3    │
│  HP條 / 屬性標籤      HP條 / MP條       │
│                                         │
│  ─────────── 戰鬥訊息欄 ───────────     │
│  「火靈妖 對 你使用了 火焰衝擊！」        │
│  「造成 128 點傷害（水屬性抗性減免 25%）」│
│                                         │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│  │ 技能  │ │ 道具  │ │ 捕捉  │ │ 逃跑  │ │
│  └──────┘ └──────┘ └──────┘ └──────┘  │
└─────────────────────────────────────────┘
```

### 2.2 隊伍編制

- **初期（單人制）**：玩家最多可帶 1 個主角 + 1 隻戰鬥魔物出征（2 vs 2 基礎配置）。
- **遭遇數量**：一般遭遇 2 隻怪物；高難度區域最多遭遇 4 隻（掉落率提升）。
- **未來擴充**：支援多玩家組隊，擴大隊伍規模。

## 三、回合流程與行動選單

### 3.1 回合流程

1. **行動順序判定**：根據場上所有角色與怪物的「速度（木屬性）」決定出手順序。
   - 每回合每個角色只能執行一次指令，無論速度差距多大。
   - 速度相同時，隨機決定先後。
2. **玩家回合**：輪到玩家角色時，從行動選單選擇指令。
3. **傷害結算**：系統根據五行相剋、抗性、暴擊率計算最終傷害。
4. **敵方回合**：怪物根據 AI 邏輯行動。
5. **戰鬥結束**：
   - 勝利：獲得經驗值、掉落素材或靈獸蛋。
   - 失敗：消耗體力值，返回地圖（或回到自宅）。

### 3.2 行動選單指令

| 指令 | 說明 |
| :--- | :--- |
| **技能** | 開啟技能列表，選擇已裝備的技能施放（消耗 MP） |
| **道具** | 開啟背包，使用恢復藥水、增益道具或捕捉符 |
| **防禦** | 本回合受到的物理與魔法傷害減半 |
| **捕捉** | 對 HP 較低的怪物使用捕捉道具，嘗試將其馴服 |
| **逃跑** | 嘗試脫離戰鬥，成功率受雙方速度差影響 |

## 四、戰鬥算法核心設計

### 4.1 傷害公式（百分比減傷制）

為避免防禦力過高導致傷害歸零，採用遞減報酬的百分比減傷公式：

```text
最終傷害 = 基礎傷害 × (1 - 減傷率) × 五行相剋係數 × 暴擊判定

其中：
基礎傷害 = 攻擊力 × 技能倍率
減傷率 = 防禦力 / (防禦力 + 100)
```

**減傷率對照表：**
- 防禦力 50 → 減傷 33%
- 防禦力 100 → 減傷 50%
- 防禦力 200 → 減傷 67%
- 防禦力 400 → 減傷 80%

### 4.2 暴擊機制（遞減報酬制）

暴擊率由火屬性決定，同樣採用遞減公式，確保永遠有提升空間且不會達到 100%：

```text
暴擊率 = 火屬性 / (火屬性 + 100) × 100%
```

- **基礎暴擊傷害**：×1.5
- **暴擊傷害上限**：×3.0（可透過裝備、技能、道具疊加）

### 4.3 五行相剋加成

| 剋制關係 | 戰鬥效果 |
| :--- | :--- |
| **木剋土** | 木屬性攻擊對土屬性怪物傷害 +25% |
| **土剋水** | 土屬性防禦對水屬性攻擊減免 +25% |
| **水剋火** | 水屬性魔攻對火屬性怪物傷害 +25% |
| **火剋金** | 火屬性攻擊對金屬性怪物傷害 +25% |
| **金剋木** | 金屬性命中對木屬性怪物穿透 +25% |

## 五、魔物 AI 行動邏輯分級

為了讓戰鬥體驗從「無腦刷怪」到「需要策略應對」有清晰的層次感，魔物 AI 分為五個等級：

| AI 等級 | 適用對象 | 行動邏輯 |
| :--- | :--- | :--- |
| **等級一** | 野生普通怪 | 完全隨機選擇技能，不判斷戰場狀況。 |
| **等級二** | 野生精英怪 | 條件觸發：HP < 50% 優先使用強化技；HP < 25% 必定使用最強技能；優先攻擊被自身屬性剋制的目標。 |
| **等級三** | 野生 Boss 怪 | 固定行動模式（如：普攻→普攻→大技→防禦循環）。HP < 30% 進入狂暴階段，行動模式改變且傷害提升。 |
| **等級四** | 道館防守 AI | 模擬玩家策略：根據被挑戰方設定的優先級執行，HP 低時優先回血，面對弱點屬性優先使用剋制技能。 |
| **等級五** | 隱藏特殊怪 | 行為不可預測：可能滿血放防禦，或具備「學習機制」（被同技能命中兩次後產生抗性）。 |

## 六、道館 PK 系統（玩家對戰）

- **介面配置**：挑戰方在右側，被挑戰方在左側。
- **防守機制**：玩家可設定一組「防守隊伍」，即使離線也能接受其他玩家的挑戰（由 AI 自動戰鬥）。
- **獎勵機制**：挑戰勝利可獲得天命幣、道館積分，或掠奪對方少部分非綁定素材。
- **賽季結算**：根據道館積分進行排名，發放賽季專屬稱號或稀有道具。
