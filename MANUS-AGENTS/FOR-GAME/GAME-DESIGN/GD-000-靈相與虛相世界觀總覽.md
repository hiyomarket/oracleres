# GD-000：靈相與虛相世界觀總覽

**版本**：v0.1（討論草稿）
**最後更新**：2026-03-23
**負責人**：遊戲 Agent
**狀態**：`草稿中` — 持續更新，尚未提交主系統審核

---

## 一、核心設計理念：雙世界架構

天命共振系統的遊戲化模組，建立在一個獨特的「雙世界架構」之上。這不僅是世界觀的設定，更是為了解決「美術成本」與「遊戲深度」之間矛盾的系統級解法。

這兩個世界分別為：**靈相世界（個人形象空間）** 與 **虛相世界（RPG 冒險世界）**。

---

## 二、靈相世界（個人形象空間）

**定位**：用戶在天命共振系統中的「真實投影」，是社交、展示與生活服務的核心介面。

### 2.1 視覺與美術風格
- **風格**：Pop Mart 盲盒公仔風（2.5-3 頭身）。
- **精細度**：極高。採用多圖層疊加技術（素體、服裝、飾品、背景獨立拆分）。
- **呈現方式**：靜態高畫質 2D 圖像，注重質感與品牌色（天命金）的融入。

### 2.2 核心功能
- **靈相空間（自宅）**：用戶的個人房間，可佈置家具、展示成就。
- **辦公室系統**：未來擴充功能，用於接取任務或進行陽宅開運等命理服務。
- **陪伴型寵物**：商城購買的貓狗等寵物，在此空間內陪伴用戶。
- **商城展示**：商城購買的道具（如神行符）在此以「精美的道具卡（類 RO 仙境傳說）」形式呈現。

---

## 三、虛相世界（RPG 冒險世界）

**定位**：用戶進行探索、戰鬥、採集與養成的遊戲化空間。

### 3.1 視覺與美術風格
- **風格**：復古像素風（類 FF6 戰鬥小人）。
- **精細度**：低。以極簡的像素點陣呈現角色與裝備差異。
- **呈現方式**：大地圖上的移動小人、橫式回合制戰鬥介面、純文字的道具列表。

### 3.2 核心功能
- **大地圖探索**：基於五行節點的資源採集與事件觸發。
- **回合制戰鬥**：與魔物進行戰鬥或捕捉。
- **裝備與道具使用**：裝備直接套用像素素材換裝；道具使用僅顯示文字提示（如：`用戶 凜冬月 使用了 神行符 ×2`），無須複雜動畫。

## 四、雙世界的身分設定與連動

為了解決玩家在兩個世界中的代入感，我們採用了「高我與執行者」的身分設定：

- **靈相世界（高我）**：這是玩家在命理層面的「高我（Higher Self）」，關注的是能量、氣場、五行平衡與美學。高我不會親自下場打怪，而是透過穿搭與佈置，將能量（祝福）傳遞給虛相世界。
- **虛相世界（執行者）**：這是玩家在物質層面的「執行者（Avatar）」，負責執行此生的任務（打怪、採集、探索）。執行者的外觀**完全取決於他身上穿的裝備與道具**，與靈相世界的時尚穿搭無關。

這種設定不僅讓世界觀更自洽，也大幅降低了美術成本——我們不需要為靈相世界的每一件時尚服裝繪製戰鬥動作，虛相世界的裝備可以獨立設計。

### 4.1 經濟閉環

靈相與虛相並非孤立存在，而是透過「資產互通」形成完整的經濟閉環：

1. **消費連動**：用戶在「靈相世界」的商城花費天命幣購買精美的道具卡，該道具會自動存入「虛相世界」的倉庫中，供冒險時使用。
2. **產出連動**：用戶在「虛相世界」中努力打怪、採集獲得的稀有素材，可以回到「靈相世界」中鍛造出獨特的展示服裝或家具。
3. **情感投射**：靈相世界滿足了用戶對「美」與「個性化」的追求（面子）；虛相世界則提供了「成長」與「策略」的遊戲樂趣（裡子）。

## 五、開發與美術資源分配策略

此架構完美解決了資源分配的難題：

- **美術 Agent 的重心**：集中於「靈相世界」的高品質紙娃娃部件，以及商城道具卡的繪製。這部分數量少但要求精緻，能直接刺激用戶消費。
- **系統 Agent 的重心**：集中於「虛相世界」的數值計算、戰鬥邏輯與龐大的道具資料庫。因為捨棄了複雜的戰鬥動畫與道具圖示，系統可以輕量化地支撐極具深度的 RPG 玩法。

## 六、靈相 ↔ 虛相 雙向牽引機制（Aura Score 與祝福）

為確保用戶每天都有動力在兩個世界之間互動，系統設計了「每日穿搭牽引機制」，將命理、時尚與 RPG 數值完美結合。

### 6.1 每日穿搭任務與 Aura Score
系統每天會根據命理曆法給出「今日五行建議」（例如：今日水能量旺盛，建議穿搭藍/黑色系）。用戶在靈相世界進行穿搭後，系統會計算出 **Aura Score（天命氣場分數，滿分 100）**：
- **主色契合度**：最高 40 分
- **飾品五行加成**：最高 20 分
- **整體協調度**：最高 20 分
- **連續穿搭天數**：最高 20 分

### 6.2 「來自靈相的祝福」隨機加成
Aura Score 不直接給予固定數值，而是決定用戶當日在虛相世界能抽到什麼等級的「隨機祝福」，創造每日驚喜感：

| 分數門檻 | 祝福等級 | 祝福效果範例（隨機抽取一項） |
| :--- | :--- | :--- |
| 0-29 分 | 無祝福 | 未完成任務或嚴重失調，無加成 |
| 30-59 分 | **普通祝福** | 今日 HP 上限 +10%、今日採集數量 +1、今日逃跑率 +30% |
| 60-79 分 | **良好祝福** | 今日全屬性 +8%、今日怪物掉落率 +15%、今日捕捉率 +20% |
| 80-100 分 | **天命祝福** | 今日全屬性 +15%、今日稀有掉落 +30%、今日可視範圍 +1 格 |

*特殊機制：連續 7 天獲得「天命祝福」，將在虛相世界解鎖隱藏天賦技能。*

### 6.3 牽引機制的商業與社群意義
- **商業帶動**：用戶為了獲得高分祝福，會主動去商城購買對應五行屬性的服裝與飾品。
- **社群帶動**：用戶會在論壇分享「今日穿搭」與「抽到的祝福」，形成高黏著度的日常話題。
