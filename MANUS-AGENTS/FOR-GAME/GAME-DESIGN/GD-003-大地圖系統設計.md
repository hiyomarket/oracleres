## 一、設計核心理念

大地圖是整個遊戲的主要活動舞台，結合真實地理位置（LBS）或虛擬網格，讓玩家在現實生活中的移動行為能夠轉化為遊戲內的探索成果。這是天命共振系統「虛實整合」的核心體現。

---

## 二、地圖形式（待確認）

目前有三個方向，需與 Boss 確認：

| 選項 | 說明 | 優點 | 缺點 |
| :--- | :--- | :--- | :--- |
| **A. 真實地圖（LBS）** | 基於 GPS 定位，玩家移動到真實地點觸發事件（如皮克敏） | 虛實整合感最強，行為可驗證 | 需要 GPS 權限，室內玩家體驗差 |
| **B. 虛擬網格地圖** | 我們自己設計一張天命世界地圖，不依賴 GPS | 完全可控，設計自由度高 | 缺乏虛實整合感 |
| **C. 混合模式（推薦）** | 虛擬地圖為主，特定地點可用真實 GPS 打卡觸發特殊事件 | 兼顧兩者優點，靈活性高 | 開發複雜度較高 |

## 三、地圖節點系統

### 3.1 節點類型

地圖上的每個節點都有主屬性，影響該節點的資源生成與加成效果：

| 節點類型 | 對應五行 | 真實地點對應 | 特殊效果 |
| :--- | :--- | :--- | :--- |
| 森林/公園節點 | 木 | 公園、植物園、山林 | 木屬性採集力 +20%，草藥類素材掉落率 + |
| 河流/海邊節點 | 水 | 河岸、海邊、湖泊 | 水屬性能量濃厚，稀有寵物出現率 + |
| 山頂/岩石節點 | 土 | 山丘、岩石地形 | 土屬性防禦加成，礦石類素材掉落率 + |
| 市場/商業節點 | 金 | 市場、商業區、百貨 | 金屬性精煉力 +，流浪商人出現機率 + |
| 廟宇/火山節點 | 火 | 廟宇、香火地、特殊地標 | 火屬性攻擊加成，技能導師出現機率 + |
| 天命節點（稀有） | 五行均衡 | 特定風水地點 | 所有五行能量 ×2，可能觸發隱藏劇情 |

### 3.2 資源生成機制（Spawning）

- **採集物**：根據時間與天氣隨機生成（雨天水屬性素材增加，晴天火屬性素材增加）
- **怪物出沒點**：特定區域（「暗能量裂隙」）定時刷新怪物
- **寵物棲息地**：某些稀有網格會出現可捕捉的靈獸
- **隱藏寶箱**：隨機生成，需要一定可視範圍才能發現

## 四、五行能量補充機制

玩家在地圖上打卡，可以獲得對應節點的五行能量：

- 在木屬性節點打卡 → 獲得木能量 +5
- 在水屬性節點打卡 → 獲得水能量 +5
- 以此類推

回到自宅後，玩家可以在「修煉室」將收集到的五行能量注入角色屬性，慢慢補足命格弱項。這是「天天上來」的核心驅動力。

## 五、NPC 系統（地圖上的特殊角色）

地圖上的 NPC 分為兩類：

### 5.1 固定 NPC

| NPC 類型 | 功能 | 位置 |
| :--- | :--- | :--- |
| 一般商人 | 販售基礎道具、技能書 | 固定在特定節點 |
| 倉庫管理員 | 管理玩家的素材倉庫 | 固定在自宅附近 |
| 修煉師 | 協助玩家注入五行能量 | 固定在修煉室 |

### 5.2 隨機 NPC（限時出現）

| NPC 類型 | 功能 | 出現規則 |
| :--- | :--- | :--- |
| **流浪商人** | 出售稀有技能書、限定道具 | 隨機地點、隨機時段，存在 2-4 小時後消失 |
| **技能導師** | 傳授高階技能（需素材 + 天命幣） | 特定節點類型出現機率較高，存在 1-2 小時 |
| **神秘旅人** | 提供隱藏任務或情報 | 極低機率出現，觸發隱藏劇情 |

> 流浪商人和技能導師的位置，只有可視範圍達到「廣域視野（3格）」以上的玩家才能看到，這讓提升水屬性成為有強烈動機的目標。

## 六、核心互動循環（一天的玩法）

| 時段 | 行為 | 消耗 | 獲得 |
| :--- | :--- | :--- | :--- |
| **早上** | 開 App，派遣角色去自動任務 | 活躍值 | 素材、五行能量（掛機收集） |
| **中午** | 收取角色帶回的素材，確認體力值 | — | 素材整理 |
| **下班路上** | 帶角色去附近地點打卡，手動採集 | 體力值 | 稀有素材、五行能量、地圖事件 |
| **晚上** | 用素材換裝/佈置自宅，派遣明天任務 | 天命幣（換裝）、活躍值 | 角色成長、自宅升級 |
