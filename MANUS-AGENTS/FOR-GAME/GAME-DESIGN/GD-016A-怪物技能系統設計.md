## 二、技能覺醒/升階機制

### 覺醒階級

```
普通 → 大招 → 奧義 → 神技
```

| 階級 | 傷害倍率 | 特效 | 稀有度 | 覺醒素材 |
|------|----------|------|--------|----------|
| 普通 | 100% | 無 | 常見 | - |
| 大招 | 150% | 特效動畫 | 稀有 | 大晶石 ×5 |
| 奧義 | 200% | 全螢幕特效 | 傳說 | 稀有素材 ×10 |
| 神技 | 300% | 專屬 CG | 天命 | 傳說素材 ×1 |

### 覺醒成功率

| 當前階級 | 目標階級 | 成功率 | 失敗懲罰 |
|----------|----------|--------|----------|
| 普通 | 大招 | 80% | 素材消失 |
| 大招 | 奧義 | 60% | 素材消失，返回普通 |
| 奧義 | 神技 | 40% | 素材消失，返回奧義 |

## 三、技能協力 Combo 系統

### Combo 觸發條件

- 同一五行屬性的技能連續使用
- 或屬性連鎖（木→土→水→火→金→木）

### Combo 效果表

| Combo 類型 | 組合範例 | 加成效果 |
|-----------|----------|----------|
| 雙連 Combo | 火 + 火 | 傷害 +20% |
| 三連 Combo | 火 + 火 + 火 | 追加燃燒（額外 30% 伤害） |
| 五行循環 | 木→土→水→火→金 | 全屬性 +10%，持續 3 回合 |
| 屬性共鳴 | 同屬性 3 個以上 | 技能冷卻 -1 回合 |
| 跨屬 Combo | 水 + 火 | 蒸發效果（伤害 ×1.5） |

### Combo 顯示

戰鬥中 Combo 發動時，畫面顯示：
```
🌟 火之共鳴！Combo ×3 傷害提升 60%！
```

## 四、寵物技能繼承機制

### 繼承方式

| 方式 | 說明 | 成功率 |
|------|------|--------|
| 捕捉繼承 | 戰鬥中捕捉怪物時有機率繼承 | 10% |
| 技能書學習 | 消耗對應技能書學習 | 100% |
| 覺醒領悟 | 寵物覺醒時自動領悟 | 100% |

### 寵物技能槽

- 寵物最多可裝備 3 個技能
- 覺醒可開啟額外技能槽（第 4 格為覺醒專屬）

### 寵物覺醒與怪物覺醒對應

```
普通寵物技能 → 覺醒後 = 怪物覺醒技能
例：藤蔓纏繞 → 覺醒 = 群體束縛
```

## 六、技能傷害數值（對應 GD-014 公式）

### 基礎傷害計算

```
技能傷害 = 基礎傷害 × 技能倍率 × (1 + 係數A) × 屬性加成 × 隨機波動

係數A = 靈力 / (靈力 + 100)
```

### MP 消耗對應

| 技能階級 | MP 消耗範圍 | 冷卻回合 |
|----------|-------------|----------|
| 初階 | 10-20 | 0-1 |
| 中階 | 20-30 | 1-2 |
| 高階 | 30-40 | 2-3 |
| 傳說 | 40-50 | 3-5 |
| 天命 | 50+ | 5+ |
