## 一、五行屬性系統重構

### 1.1 天命共振 vs 魔力寶貝 屬性對照

| 天命共振（五行） | 魔力寶貝（四屬性） | 對應關係 | 戰鬥定位 |
| :--- | :--- | :--- | :--- |
| 木 | 風 | 木→土對應風→地 | 智力/採集/治癒/速度 |
| 火 | 火 | 同名直接對應 | 力量/鍛冶/攻擊/暴擊 |
| 土 | 地 | 土被木剋（木→土）| 體魄/承重/防禦/HP |
| 金 | 金屬系（種族非屬性）| 對應金屬系種族 | 敏捷/精煉/命中/閃避 |
| 水 | 水 | 同名直接對應 | 精神/尋寶/魔攻/異常觸發 |

> 設計差異：魔力寶貝只有4屬性，天命共振有5屬性，多出木與金。
> 木對應魔力寶貝的風；金在魔力寶貝裡是種族（金屬系），需另外建模。


## 二、五行水晶點數制

### 2.1 點數制設計

每位角色根據命盤五行比例分配 100 點作為初始五行水晶。
命盤木42% + 水35% + 火11% + 土9% + 金4% = 角色初始木42點/水35點/火11點/土9點/金4點

### 2.2 點數與剋制效果對應

| 每點屬性值 | 全剋效果（每點） | 半剋效果（每點） |
| :--- | :--- | :--- |
| 1點 = 0.3（全剋係數）| 剋制屬性 × 0.3% | 半剋 × 0.15% |

推導：魔力寶貝10點 × 每點3% = 30% → 天命共振1點 = 0.3% × 剋制方屬性值

### 2.3 水晶類型

| 水晶類型 | 點數組合 | 特性 |
| :--- | :--- | :--- |
| 純水晶 | 100單一屬性 | 最強單屬性攻擊 |
| 相剋水晶 | 例：60木40土 | 進攻型（剋60，被剋40）|
| 相生水晶 | 例：60火40木 | 均衡型（火生木）|
| 無關水晶 | 例：50木50水 | 防禦型（木水無生剋）|
| 全五行水晶 | 每種20點 | 完全無剋制（終極防禦）|

## 三、物理傷害公式

### 3.1 攻防閾值240理論（核心改動）

重要調整：舊公式減傷率 = DEF/(DEF+100) 改為240閾值遞減報酬。

| 面板數值 | 實際效應值 |
| :--- | :--- |
| ≤ 240 | 直接等於面板數值 |
| > 240 | 240 + 30% × (面板數值 - 240) |

實例對照：

| 面板防禦 | 舊減傷率 | 新實際效應值 | 舊受傷 | 新受傷 |
| :--- | :--- | :--- | :--- | :--- |
| 50 | 33% | 50 | 67% | 67% |
| 100 | 50% | 100 | 50% | 50% |
| 200 | 67% | 200 | 33% | 33% |
| 240 | 71% | 240 | 29% | 29% |
| 300 | 75% | 258 | 25% | 29% |
| 400 | 80% | 288 | 20% | 29% |
| 1000 | 91% | 468 | 9% | 11% |

> 舊公式缺點：防禦越高，堆防收益急降但永不吃虧，高防角色近乎無敵。
> 新公式優點：超240後每投入100點面板防禦，實際防禦只增加30點，防止數值膨脹失控。

### 3.2 中值傷害公式

數學上與魔力寶貝完全等價：

```
天命共振：中值傷害 = ATK² / (ATK/3 + DEF)
魔力寶貝：中值傷害 = 3×ATK² / (ATK + 3×DEF)
```

兩者數學等價，差異僅在表達形式。

### 3.3 傷害波動（90% ~ 110%）

每次物理攻擊隨機抽取21種倍率之一：
90%, 91%, 92%, 93%, 94%, 95%, 96%, 97%, 98%, 99%, 100%, 101%, 102%, 103%, 104%, 105%, 106%, 107%, 108%, 109%, 110%

均值為100%，高低各10%浮動。

### 3.4 武器加成

| 武器狀態 | 效果 |
| :--- | :--- |
| 攜帶武器（任何） | 傷害 ×1.15（+15%）|
| 弓箭/法杖額外 | 另有技能倍率加成 |
| 格鬥自帶 | 無需武器即有15%加成 |

## 四、魔法傷害公式

### 4.1 五行精神（MND）門檻表

| 魔法等級 | 等級所需精神 | 單體法術（基礎）| 強法術（基礎）| 超法術（基礎）|
|:---:|:---:|:---:|:---:|:---:|
| 1 | 103 | 95 | 57 | 38 |
| 2 | 123 | 171 | 103 | 68 |
| 3 | 143 | 239 | 143 | 96 |
| 4 | 163 | 303 | 182 | 121 |
| 5 | 183 | 368 | 221 | 147 |
| 6 | 203 | 432 | 259 | 173 |
| 7 | 223 | 497 | 298 | 199 |
| 8 | 243 | 570 | 342 | 228 |
| 9 | 263 | 646 | 388 | 258 |
| 10 | 283 | 722 | 433 | 289 |

> 等級所需精神公式：103 + 20 × 等級，與魔力寶貝完全一致。

### 4.2 精神比例係數A（對魔時）

精神比例 = (自身水屬性 / 敵方水屬性) × 100

| 精神比例 | 係數A（角色）| 說明 |
|:---:|:---:|:---|
| > 119% | 1.1 | 顯著優勢 |
| 113~119% | 1.0 | 基本持平 |
| 105~113% | 0.9 | 輕微劣勢 |
| 97~105% | 0.7 | 劣勢 |
| 89~97% | 0.6 | 劣勢明顯 |
| 79~89% | 0.4 | 劣勢很大 |
| 69~79% | 0.3 | 幾乎打不動 |
| ≤ 69% | 0.1 | 完全打不動 |

> 法師內戰關鍵：精神比例低於69%時，係數只剩0.1！高水屬性法師對低水屬性法師有壓制性優勢。

### 4.3 係數B（五行精神滿足度）

係數B = 0.8 × (自身水屬性 / 等級所需精神)  [範圍：0 ~ 0.8]

水屬性達到門檻 → 係數B = 0.8（最高）

### 4.4 係數C（五行魔攻滿足度）

係數C = (自身水屬性魔攻 - 等級所需魔攻) / 等級 / 10  [範圍：0 ~ 0.4]

等級所需魔攻：10 + 30 × (等級 - 1)

| 等級 | 等級所需魔攻 |
|:---:|:---:|
| 1 | 10 |
| 5 | 130 |
| 10 | 280 |

### 4.5 法術傷害公式

法術傷害 = 法術基本值 × 係數A × (係數B + 係數C)

實例（Lv10，水屬性280，敵水屬性200，魔攻150，Lv10單體）：
```
精神比例 = 280/200 × 100 = 140% → 係數A = 1.1
係數B = 0.8 × (280/303) = 0.739
係數C = (150-280)/10/10 → 強制為 0
法術傷害 = 722 × 1.1 × (0.739 + 0) ≈ 587（中值）
```

### 4.6 五行法術屬性標籤

| 法術屬性 | 剋制目標 |
| :--- | :--- |
| 木 | 土屬性敵人 |
| 火 | 金屬性敵人 |
| 土 | 水屬性敵人 |
| 金 | 木屬性敵人 |
| 水 | 火屬性敵人 |
| 無 | 無剋制加成 |

## 五、五行相剋與屬性加成

### 5.1 全剋與半剋判斷邏輯

全剋：攻方為純屬性（100點單一屬性）vs 受方為純屬性
半剋：攻守雙方皆為複合屬性時，屬性差值產生半剋效果
無關：攻守為無生剋關係的複合屬性（如木60水40 vs 金60水40）

### 5.2 屬性加成完整公式

```
屬性加成% = 100 + 30 × [
    (我木×敵土 + 我火×敵金 + 我土×敵水 + 我金×敵木 + 我水×敵火)
  - (我土×敵木 + 我金×敵火 + 我水×敵土 + 我木×敵金 + 我火×敵水)
] / 100
```

> 與魔力寶貝四屬性公式（去掉風→地，新增木→金→水→木→火→金迴路）等價推廣。

### 5.3 實戰範例

範例一：純火（100火）攻 純金（100金）
火→金全剋：對敵+30%傷 / 受敵-20%傷

範例二：60火40木 攻 50金50水
火→金：40×50×0.3 = +6%
木→金：40×50×0.3 = +6%
金→木：50×40×0.3 = +6%（反向剋制）
水→火：50×50×0.3 = +7.5%
最終加成 ≈ +25%（綜合計算）

範例三：木60水40 vs 金60木40
金→木：60×60×0.3 = +10.8%
木→水：無剋制關係 → 0
水→木：無剋制關係 → 0
最終加成 ≈ +10.8%

### 5.4 複合屬性半剋量化表

| 複合比例 | vs 純屬性 | 效果 |
| :--- | :--- | :--- |
| 100:0（純）| vs 純 | 全剋 +30% |
| 80:20 | vs 純 | +24% |
| 60:40 | vs 純 | +18% |
| 50:50 | vs 純 | 半剋 +15% |
| 40:60 | vs 純 | +9% |
| 20:80 | vs 純 | +6% |

| 抗五行值 | 所屬級別 | 區間範圍 |
|:---:|:---:|:---|

| 火屬性 | 天命一击率 |

| 50 | 33.3% |
| 70 | 41.2% |

### 7.2 天命一击額外傷害

天命一击額外傷 = floor(敵方面板防禦 × 攻方等級 / 敵方等級 / 2)
天命一击總傷 = 天命一击額外傷 + 普通中值傷害

## 九、完整傷害計算流程

### 9.1 物理傷害完整流程（9步）

```
步驟1：面板數值（木屬性攻擊力/金屬性命中/土屬性防禦）

步驟2：240閾值遞減
if 面板 > 240:
    實際值 = 240 + 30% × (面板 - 240)
else:
    實際值 = 面板

步驟3：中值傷害
中值 = ATK² / (ATK/3 + DEF)

步驟4：傷波
中值 × rand(90%~110%)

步驟5：武器加成
× 1.15（如有攜帶武器）

步驟6：五行屬性加成
× (100 + 木剋土% + 火剋金% + 土剋水% + 金剋木% + 水剋火%) / 100

步驟7：種族加成
× (100 + 種族剋制%) / 100

步驟8：天命一击加成
+ floor(敵DEF × 我LV / 敵LV / 2)（如有觸發天命一击）

步驟9：技能倍率
× 技能倍率

最終傷害 = max(普通物理傷害, 天命一击總傷)
最低為 1
```

### 9.2 魔法傷害完整流程（8步）

```
步驟1：法術基本值（查表：單體/強/超）

步驟2：精神比例
自身水屬性 / 敵方水屬性 × 100 → 查係數A表

步驟3：係數B
係數B = 0.8 × (自身水屬性 / 等級所需精神)

步驟4：係數C
係數C = (自身魔攻 - 等級所需魔攻) / 等級 / 10

步驟5：基礎法術傷害
基礎 = 法術基本值 × 係數A × (係數B + 係數C)

步驟6：傷波
× rand(95%~105%)

步驟7：五行屬性加成 + 種族加成
× (100 + 五行剋制% + 相生光環% + 種族剋制%) / 100

步驟8：抗五行減傷（隨機）
最終 = 基礎 × (100 - 隨機減傷率) / 100

最低為 1
```
