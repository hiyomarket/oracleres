## 一、設計核心理念

天命共振的技能系統沒有職業限制。每位用戶的命格決定起點，但透過在大地圖上持續收集五行能量、購買技能書、遭遇隨機 NPC，任何人都能慢慢補足弱項，打造出屬於自己的獨特流派。

技能槽的數量有限（由木屬性/智力決定），這讓「技能搭配」本身成為一種策略，也讓玩家有持續探索與優化的動力。

## 二、技能分類架構

### 2.1 戰鬥系技能

| 子分類 | 說明 | 範例技能 |

| 子分類 | 說明 | 範例技能 |

### 2.4 隱藏天賦技能（成就型，不佔技能槽）

這類技能解鎖後永久生效，不佔用技能槽，性質等同於「成就解鎖」。觸發條件對玩家部分或全部隱藏，是遊戲社群討論的核心驅動力。

## 三、技能獲取方式

### 3.1 商店購買（公開技能）

- **一般商店**：販售基礎技能書，所有玩家皆可購買。
- **高級商店**（需達到特定等級才能進入）：販售進階技能書。

### 3.2 隨機 NPC 觸發（限時技能）

地圖上會隨機出現兩種特殊 NPC，存在時間有限制，錯過即消失：

- **流浪商人**：出售稀有技能書，每次出現的商品隨機，價格較高。
- **技能導師**：可用「特定素材 + 天命幣」換取高階技能傳授，每位導師只教一個技能。

### 3.3 天賦觸發（隱藏技能）

當角色滿足隱藏條件時，系統自動解鎖技能，不需要任何主動操作。

## 四、隱藏天賦技能：多條件觸發機制

### 4.1 核心規則

每個隱藏天賦技能有 **2-3 個觸發條件**，必須同時滿足才能解鎖。

技能解鎖後，角色資料頁顯示規則：
- **已達成的條件**：顯示玩家實際滿足的那一條（或多條）
- **未達成的條件**：全部顯示為「？？？」

> 同一個技能，不同玩家可能因為不同的條件組合而觸發，製造論壇討論熱度。

### 4.2 可用的觸發條件類型

| 條件類型 | 範例 |
| :--- | :--- |
| **屬性數值** | 水屬性達到 80 |
| **行為累積** | 累計打卡 100 次、累計採集 500 次 |
| **命理資料（獨家）** | 日主為甲木、用神為火、生於冬季（亥子丑月）、子時出生 |
| **特殊日期** | 生日當天登入、天命吉日登入 |
| **命格特徵** | 命盤中有「天乙貴人」、某五行比例超過 50% |
| **跨系統行為** | 在主系統完成 XX 次命理解讀（將遊戲與核心功能連結） |
| **持有條件** | 持有特定寵物、裝備特定道具 |
| **連續行為** | 連續登入 30 天、連續 7 天完成採集任務 |

## 五、技能槽系統

- **技能槽數量**：由「木屬性（智力）」決定，智力每 20 點解鎖一個槽，最多 10 槽。
- **隱藏天賦技能不佔槽**：成就型技能永久生效，不佔用技能槽空間。
- **技能搭配自由**：玩家可以在任何時候調整技能槽的配置，製造不同的流派玩法。

## 六、自由流派範例（無職業限制的深度）

| 流派名稱 | 核心屬性 | 玩法特色 |
| :--- | :--- | :--- |
| **天命術士** | 高木 + 高水 | 廣域探索 + 高魔攻 + 超強採集，帶水屬性寵物，地圖幾乎全透明 |
| **鋼鐵行者** | 高土 + 高金 | 超高防禦 + 命中滿格，專門刷高等怪物的稀有裝備 |
| **萬能工匠** | 高火 + 高金 | 鍛造 + 精煉雙滿，不打怪，靠製造裝備賺取天命幣 |
| **幻影刺客** | 後天補水 + 金 | 高閃避 + 暴擊 + 毒，一擊必殺流，需刻意補足弱項才能養成 |

## 七、後台技能管理規格（供主系統 Agent 參考）

技能資料庫應設計為可動態新增的活資料表，每筆技能包含：

```
- 技能名稱
- 技能分類（戰鬥/生活/生產/隱藏天賦）
- 效果描述（顯示給玩家）
- 效果數值（後台調整用，不顯示給玩家）
- 技能槽類型（佔槽 / 不佔槽）
- 觸發條件 1-3（各自可設定「顯示」或「隱藏」）
- 觸發條件的邏輯類型（屬性值 / 命理資料 / 行為累積 / 日期 / 持有條件）
- 是否啟用（可隨時上下架）
- 備註（內部說明，不顯示給玩家）
```
