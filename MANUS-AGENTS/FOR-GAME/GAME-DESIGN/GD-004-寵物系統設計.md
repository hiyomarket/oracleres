## 一、設計核心理念

寵物系統分為兩大類型，各有不同的核心功能。寵物是補足玩家弱項的重要手段，也是情感投射的延伸。

---

## 二、寵物雙系統

### 2.1 隨行型寵物（探索夥伴）

跟著主人在大地圖上移動，提供被動加成，不參與戰鬥。核心價值是「補足主人的弱點」。

| 寵物屬性 | 提供的被動加成 |
| :--- | :--- |
| 水屬性靈獸 | 可視範圍 +1 格、稀有素材掉落率 + |
| 木屬性靈獸 | 採集速度加快、自動識別藥草 |
| 火屬性靈獸 | 移動速度 +、遭遇怪物時先制攻擊 |
| 土屬性靈獸 | 背包承重 +、採集點刷新速度加快 |
| 金屬性靈獸 | 素材品質提升、精煉成功率 + |

**獲取方式**：在地圖上特定棲息地遭遇並捕捉（需消耗「靈獸誘餌」道具）

### 2.2 戰鬥型寵物（戰場夥伴）

進入戰鬥後才召喚，擁有自己的技能和屬性，可以和主人形成「組合技」。

**獲取方式**：擊敗精英怪物後有機率掉落「靈獸蛋」，孵化後獲得

---

## 三、寵物屬性與成長

- 寵物有自己的五行屬性值
- 玩家可以用五行能量「餵養」寵物，提升其屬性
- 寵物也有等級系統，等級越高加成越強
