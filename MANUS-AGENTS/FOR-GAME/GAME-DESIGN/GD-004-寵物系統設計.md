## 一、設計核心理念

本系統採用「魔物 = 寵物」的統一架構，玩家在地圖上遭遇的魔物，經過捕捉後即成為寵物。這不僅大幅降低開發與美術成本，也讓遊戲世界觀更加自洽。

寵物系統分為兩個截然不同的層次：
1. **魔物型寵物**：透過戰鬥捕捉，具有五行屬性與戰鬥能力，是玩家探索與戰鬥的夥伴。
2. **陪伴型寵物**：透過商城購買（如貓、狗），無戰鬥能力，純粹提供情感陪伴與外觀展示。

---

## 二、魔物型寵物（戰鬥與探索夥伴）

### 2.1 狀態轉換

| 狀態 | 名稱 | 說明 |
| :--- | :--- | :--- |
| **野生** | 魔物 | 在地圖上出沒，玩家可選擇「戰鬥擊敗」或「嘗試捕捉」 |
| **馴服** | 寵物 | 被捕捉後，可編入隊伍出征，或設定為隨行狀態提供被動加成 |

### 2.2 捕捉機制

- **條件**：魔物 HP 低於一定比例時，才能使用捕捉指令。
- **消耗**：需消耗「靈獸誘餌」或「捕捉符」等道具。
- **成功率影響因素**：
  - 玩家的水屬性（精神力）越高，成功率越高。
  - 捕捉道具的品質（普通/高級/傳說）。
  - 魔物剩餘 HP 越低，成功率越高。

---

## 三、魔物五維成長系統（核心養成機制）

為了增加重複捕捉的趣味性與社群討論度，每隻魔物都擁有隱藏的成長數值。

### 3.1 五維屬性定義

| 維度 | 對應五行 | 說明 |
| :--- | :--- | :--- |
| **生命（HP）** | 土 | 最大血量 |
| **攻擊** | 火 | 物理傷害基礎值 |
| **防禦** | 金 | 物理傷害減免 |
| **魔力** | 水 | 魔法技能強度與 MP 上限 |
| **速度** | 木 | 行動順序，速度高者先手 |

### 3.2 成長率機制（隱藏數值）

每隻魔物在被捕捉的瞬間，系統會隨機生成一組「成長率」，對玩家完全隱藏，只能在升級過程中觀察得知。

| 成長率等級 | 每級提升幅度 | 稀有度 |
| :--- | :--- | :--- |
| **普通** | 每級 +1~2 | 常見 |
| **良好** | 每級 +2~3 | 不常見 |
| **優秀** | 每級 +3~5 | 稀有 |
| **天賦** | 每級 +5~8 | 極稀有 |
| **傳說** | 每級 +8~12 | 幾乎不存在 |

> **設計目的**：同種魔物在高等級時，傳說成長率的個體能力可能是普通個體的數倍，這是驅動玩家不斷重複捕捉的核心動力。

### 3.3 個體值系統（更細緻的隱藏數值）

在成長率之外，五個維度各有一個「個體值」（0-31），決定該維度的天花板。
- 個體值高的維度，在成長率觸發時有機會獲得額外加成。
- 玩家可使用「鑑定符」道具查看某維度的個體值範圍（高/中/低），但無法得知精確數字。

### 3.4 捕捉品質的影響因素

| 影響因素 | 說明 |
| :--- | :--- |
| **玩家水屬性** | 水屬性（精神力）越高，捕捉到「優秀/天賦」成長率的機率越高 |
| **捕捉道具** | 道具品質越高，獲得高成長率的機率越好 |
| **特殊時間點** | 在五行能量最高峰的日子捕捉，有機率觸發「天賦個體」 |
| **命格共鳴** | 玩家命格與魔物五行屬性相符時，捕捉品質有額外加成 |

## 四、陪伴型寵物（情感與生活夥伴）

- **定位**：純粹的情感陪伴與外觀展示，無戰鬥能力，無五行屬性。
- **獲取方式**：透過商城使用天命幣購買（如各種品種的貓咪、狗狗、小鳥等）。
- **功能**：
  - 居住在玩家的「角色空間/自宅」中。
  - 可設定為「隨行」，在大地圖上跟隨玩家角色移動。
  - 可進行簡單互動（餵食、撫摸），增加親密度，解鎖特殊動作或表情。

## 五、特殊活動企劃範例：五行靈獸降臨

結合天命共振系統獨有的命理曆法，舉辦限時捕捉活動：

> **活動公告範例**：
> 「今日為庚申日，金屬性能量達到本月最高峰！請各位天命者帶著『高級靈獸符』前往地圖上的『市場/商業節點』，將有極高機率遭遇並捕捉到擁有『傳說成長率』的稀有金靈獸！」

這種活動能有效結合現實時間、地理位置與遊戲機制，創造極大的社群話題與參與度。
