## 補充一：魔法傷害精神比例公式修正

### 問題說明

GD-005 整合版將魔力寶貝的「精神（MND）」直接對應至天命共振的「水屬性」，導致魔法係數 A 的公式為：

```
精神比例 = 自身水屬性 / 敵方水屬性
```

此設計造成命盤以火、木、土、金為主的玩家，水屬性偏低（例如 8%），對上水屬性高的敵人時，係數 A 可能只有 0.1，魔法傷害幾乎歸零。這與天命共振的核心設計矛盾——每種命盤都應有其戰鬥價值，不應因命盤屬性不同而讓某類玩家完全無法使用魔法。

### 修正方案

**改採「技能屬性 vs 敵方對應抗性」的比較邏輯**，而非固定用水屬性比較。

#### 修正後的係數 A 計算邏輯

```
施法者的技能屬性值 = 施法者對應五行屬性 × 100（標準化至100分制）
敵方對應抗性值 = 敵方對應五行抗性（詳見 GD-005 第十二章）

精神比例 = (施法者技能屬性值 / 敵方對應抗性值) × 100%
```

#### 係數 A 對照表（維持魔力寶貝原始結構）

| 精神比例 | 係數 A（角色）| 係數 A（靈寵）|
|:---:|:---:|:---:|
| > 119% | 1.1 | 1.1 |
| 113% ~ 119% | 1.0 | 1.1 |
| 105% ~ 113% | 0.9 | 1.05 |
| 97% ~ 105% | 0.7 | 0.95 |
| 89% ~ 97% | 0.6 | 0.9 |
| 79% ~ 89% | 0.4 | 0.7 |
| 69% ~ 79% | 0.3 | 0.6 |
| ≤ 69% | 0.1 | 0.4 |

#### 實例說明

**火屬性玩家（火 45%）使用「炎爆術」攻擊土屬性怪物（土抗性 30%）：**

```
施法者技能屬性值 = 45（火屬性 45%，標準化）
敵方對應抗性值 = 30（土系怪物的爆炸/範圍抗性，對應火）
精神比例 = 45 / 30 × 100% = 150%
係數 A = 1.1（最高檔）
```

**水屬性玩家（水 8%）使用「寒冰術」攻擊同一怪物：**

```
施法者技能屬性值 = 8（水屬性 8%）
敵方對應抗性值 = 25（土系怪物的魔法抗性，對應水）
精神比例 = 8 / 25 × 100% = 32%
係數 A = 0.1（最低檔）
```

**結論**：火屬性玩家使用火系魔法效果最佳；水屬性低的玩家不應使用水系魔法，但可以使用其他屬性的技能。這樣的設計強化了「用自己命盤強項打仗」的核心體驗。

## 補充二：命盤五行比例 → 屬性數值換算公式

### 問題說明

GD-005 與 GD-018 均提及五行屬性影響角色數值，但未定義「命盤比例如何轉換為具體數值」的統一公式，導致實作時可能出現不一致。

### 統一換算公式

#### 基礎屬性計算（以 Lv.1 為基準）

所有屬性的計算遵循以下統一結構：

```
屬性基礎值 = 基礎常數 + 等級加成 + 命盤加成

命盤加成 = 對應五行百分比 × 命盤加成係數
```

#### 各屬性換算表

| 屬性 | 對應五行 | 基礎常數 | 等級加成 | 命盤加成係數 | Lv.1 範例（五行50%） |
|:---|:---:|:---:|:---:|:---:|:---:|
| HP | 木 | 100 | +10/Lv | ×3.0 | 100 + 0 + 150 = **250** |
| 物理攻擊（ATK）| 火 | 10 | +2/Lv | ×1.5 | 10 + 0 + 75 = **85** |
| 物理防禦（DEF）| 土 | 10 | +1.5/Lv | ×1.5 | 10 + 0 + 75 = **85** |
| 速度（SPD）| 金 | 5 | +0.5/Lv | ×0.8 | 5 + 0 + 40 = **45** |
| 魔法攻擊（MATK）| 水 | 10 | +2/Lv | ×1.5 | 10 + 0 + 75 = **85** |
| MP | 水 | 50 | +5/Lv | ×1.5 | 50 + 0 + 75 = **125** |

> 注意：水屬性同時影響 MATK 和 MP，兩者使用相同的命盤加成係數但基礎常數不同。

#### 典型命盤範例

以「木命為主（木 42%、火 25%、土 20%、金 8%、水 5%）」的 Lv.1 旅人為例：

| 屬性 | 計算式 | 結果 |
|:---|:---|:---:|
| HP | 100 + 0 + 42×3.0 | **226** |
| ATK | 10 + 0 + 25×1.5 | **48** |
| DEF | 10 + 0 + 20×1.5 | **40** |
| SPD | 5 + 0 + 8×0.8 | **11** |
| MATK | 10 + 0 + 5×1.5 | **18** |
| MP | 50 + 0 + 5×1.5 | **58** |

> 此玩家 HP 高、攻防均衡，但速度慢、魔法弱——完全符合「木命」的生命力旺盛特質。

#### 與 GD-018 的對齊說明

GD-018 定義了「等級扁平化」原則：等級加成幅度小，命盤加成才是主要差異來源。本換算公式完全遵循此原則——等級從 1 升到 60，HP 僅增加 590（+10×59），但命盤木屬性從 20% 提升到 50%，HP 可增加 90（30%×3.0），兩者量級相當，確保等級不會壓過命盤的影響力。

## 補充三：放置遊戲戰鬥日誌簡化呈現規格

### 問題說明

GD-005 的完整戰鬥公式（9 步物理、8 步魔法）是為回合制即時戰鬥設計的。在放置型文字遊戲中，玩家無法即時操作，日誌是唯一的戰鬥體驗媒介，因此需要一套「簡化呈現規格」——底層仍用完整公式計算，但日誌文字只顯示關鍵資訊。

### 日誌呈現格式規範

#### 標準戰鬥日誌（每回合）

```
[旅人名稱] 對 [怪物名稱] 發動 [技能/普攻]！
→ [五行加成說明（如有）]
→ 造成 [傷害數字] 點傷害[（天命一击！）]
→ [怪物名稱] 剩餘 HP：[當前HP] / [最大HP]
```

**實例：**
```
旅人「陳木靈」對「九份金礦精靈」發動普通攻擊！
→ 木剋土，傷害提升 30%
→ 造成 142 點傷害（天命一击！×1.8）
→ 金礦精靈 剩餘 HP：58 / 200
```

#### 特殊事件日誌

| 事件類型 | 日誌格式 |
|:---|:---|
| 五行全剋觸發 | `[五行A] 剋 [五行B]，傷害提升 30%！` |
| 五行半剋觸發 | `[五行A] 微剋 [五行B]，傷害提升 15%` |
| 天命一击（暴擊） | `天命一击！傷害 ×[倍率]` |
| 抗魔減傷觸發 | `[怪物名稱] 的靈力抵擋了部分傷害（減傷 [X]%）` |
| 種族剋制觸發 | `[種族A] 剋制 [種族B]，傷害提升 [X]%！` |
| 旅人被擊敗 | `旅人 [名稱] 力竭倒地，在 [地點] 休息恢復中...` |
| 怪物擊敗 | `[怪物名稱] 被擊敗！獲得 [EXP] 經驗、[金幣] 金幣[、[道具]（如有掉落）]` |

#### 戰鬥摘要（每次戰鬥結束後）

```
【戰鬥結束】
地點：[節點名稱]
對手：[怪物名稱]（[五行屬性] / [種族]）
結果：[勝利/失敗]
總傷害：[數字] | 天命一击次數：[數字]
獲得：[EXP] 經驗 / [金幣] 金幣 / [道具（如有）]
```

#### 不應顯示的資訊

以下計算過程**不應出現在日誌中**，僅在後台計算：

- 係數 A、B、C 的具體數值
- 240 閾值遞減後的實際攻防數值
- 抗魔萬分表的隨機抽取過程
- 傷害波動的具體倍率（90%~110%）

## 補充四：台灣地圖怪物種族對應表

### 問題說明

GD-005 定義了 10 種族，但現有 50 隻怪物（shared/monsters.ts）的種族分配未明確定義，導致種族剋制系統無法正確運作。

### 怪物種族對應原則

種族分配依據怪物的**外觀形態**與**五行屬性**雙重判斷：

| 種族 | 對應怪物類型 | 五行傾向 |
|:---|:---|:---|
| 靈獸系 | 動物形態的靈體（山豬精、鹿靈等）| 木、土 |
| 亡魂系 | 幽靈、怨靈、不死類（陰靈、冤魂等）| 水、金 |
| 金屬系 | 礦石精靈、機械類（金礦精靈、石傀儡等）| 金、土 |
| 人型系 | 人形妖怪、修煉中的精怪 | 全屬性均有 |
| 植物系 | 樹精、藤蔓類（竹林精靈、榕樹老怪等）| 木 |
| 水生系 | 海洋、河流類（海龍王、水鬼等）| 水 |
| 妖化系 | 動物修煉成精（狐妖、蛇妖等）| 火、木 |
| 龍種系 | 龍類、蛟類（蛟龍、山龍等）| 土、水 |
| 蟲類系 | 昆蟲類（蜈蚣精、蜘蛛妖等）| 木、金 |
| 天命系 | Boss 專屬（五行守護神、地脈龍王等）| 複合屬性 |

### 種族剋制在台灣地圖的設計意圖

台灣地圖的怪物種族分布並非隨機，而是依據地理特色刻意設計：

- **北部山區**：植物系、妖化系為主 → 剋制這些種族的玩家（不死系、靈獸系）在北部探索更有優勢
- **中部高山**：龍種系、靈獸系為主 → 飛行系、特殊系玩家的主場
- **南部沿海**：水生系、人型系為主 → 植物系、不死系玩家的主場
- **離島**：龍種系 Boss → 需要飛行系寵物才能有效剋制
