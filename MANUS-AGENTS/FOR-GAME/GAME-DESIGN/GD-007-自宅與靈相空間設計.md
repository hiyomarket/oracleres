## 一、系統定位

自宅是「靈相世界」的核心，是用戶在天命共振系統裡的「個人基地」。它不僅是展示高品質紙娃娃（靈相角色）的空間，更是連結命理服務、時尚養成與虛相冒險的樞紐。

---

## 二、v1.0 核心實作範圍（初期目標）

為確保開發資源的有效運用，v1.0 版本專注於「靈相角色的動態展示與換裝體驗」，其餘功能列為未來擴充。

### 2.1 靈相角色動態展示
- **三視角切換**：角色具備「正面（0°）」、「左側 45°」、「右側 45°」三個視角的完整圖層素材。
- **基礎動態感**：透過前端 CSS transition 在三個視角間切換，配合簡單的位移，營造角色在房間內走動、轉身的「活著」的感覺。
- **狀態反映**：若用戶連續多日未登入，角色會呈現「無精打采」的待機狀態，產生情感牽引。

### 2.2 換裝系統
- **圖層疊加**：支援素體、服裝（上衣/下衣/鞋子）、飾品（頭飾/項鍊/戒指/手環）、髮型的多圖層疊加。
- **每日穿搭任務**：用戶在此介面完成穿搭，系統計算 Aura Score 並給予「來自靈相的祝福」（詳見 GD-000）。

## 三、未來擴充功能（v2.0 及以後）

隨著系統穩定，自宅將逐步解鎖以下空間與功能：

| 空間名稱 | 擴充功能說明 |
| :--- | :--- |
| **起居室** | 開放「陪伴型寵物」在此自由活動；開放裝飾性家具佈置。 |
| **修煉室** | 允許將虛相世界採集的「五行能量」注入角色屬性，補足命格弱項。 |
| **倉庫** | 存放商城購買的道具，並可放置「功能性家具」（如：強化鍛冶台，提升鍛造成功率）。 |
| **辦公室** | 接取命理服務任務、預約命理師諮詢；開放「陽宅開運服務」（用戶設定自家格局，系統給出五行開運建議）。 |
| **庭院/農場** | 種植從虛相世界帶回的「五行種子」，植物生長速度與每日 Aura Score 連動。 |

## 四、美術素材需求（給美術 Agent 的規格）

為實現 v1.0 的動態展示，美術 Agent 需在現有 TASK-003（正面視角）的基礎上，擴充以下規格：

1. **新增視角**：左側 45°、右側 45°。
2. **圖層拆分**：每個新視角皆需嚴格遵守 TASK-003 的 7 大圖層拆分規範（素體、服裝各部位、飾品各部位獨立輸出）。
3. **對齊基準**：三個視角的素體中心點必須絕對對齊，以確保前端切換視角時不會產生不自然的跳動。
4. **背景透明**：所有輸出的 PNG 必須確保絕對透明背景。
