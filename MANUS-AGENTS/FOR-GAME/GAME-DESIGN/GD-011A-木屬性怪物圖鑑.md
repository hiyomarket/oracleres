## 📌 欄位說明（對應 DB-SCHEMA Monster DB）

| 欄位 | 說明 |
|:---|:---|
| monster_id | 系統流水號 |
| monster_code | 程式用 ID（如 M_W001）|
| 五行 | 木（固定）|
| 等級 | 1–60 |
| 稀有度 | 普通/精英/Boss/世界Boss |
| HP / 攻 / 防 / 速度 | 基礎屬性 |
| 命中力 | 影響攻擊命中率 |
| 魔法攻擊 | 法術技能威力 |
| 抗木/火/土/金/水% | 五維抗性（0–90）|
| 被剋制加成% | 屬性被剋時額外承受傷害（0–50）|
| skill_id_1/2/3 | 怪物技能（關聯 MonsterSkill DB）|
| AI 等級 | 1–5（行為邏輯）|
| 成長率 | 普通/良好/優秀/天賦/傳說 |
| 掉落素材 | 對應 Item DB |
| 掉落機率% | 基礎掉落機率 |

## 六、戰鬥結算說明（對應 GD-005）

### 6.1 屬性剋制計算
```
木剋土 → 火剋金 → 土剋水 → 金剋木 → 水剋火
被剋制時：額外承受 = 技能傷害 × (怪物被剋制加成% / 100)
```

### 6.2 五維抗性計算
```
實際受到屬性傷害 = 技能基礎傷害 × (1 - 抗性% / 100)
抗性上限：90%
```

### 6.3 AI 等級行為邏輯
| AI 等級 | 怪物類型 | 行為模式 |
|:---:|:---|:---|
| 1 | 普通怪 | 完全隨機普攻 |
| 2 | 精英怪 | HP<50% 使用強化技能 |
| 3 | Boss | 固定循環：普攻→普攻→大招→防禦 |
| 4 | 道館防守 | 根據敵方屬性選擇剋制技能 |
| 5 | 世界Boss/隱藏 | 不可預測，可能預判玩家行動 |
