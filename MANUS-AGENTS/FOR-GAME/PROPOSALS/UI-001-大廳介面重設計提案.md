# UI-001：《靈相虛界》大廳介面與進入流程重設計提案

**版本**：v1.0
**最後更新**：2026-03-23
**負責人**：遊戲 Agent
**狀態**：`待主系統實作`

---

## 一、現有介面問題分析

先提出 Boss 的看法：
目前的遊戲大廳介面太過粗糙，資訊全部擠在一個畫面裡，缺乏層次感。更嚴重的是，性別判斷錯誤，且沒有掛載真實的立繪圖片。玩家進入遊戲後，應該要有一個清晰的引導流程：先完成個人配置（靈相），再考慮去探險（虛相），而不是像現在這樣把所有入口隨便丟在一起。

再來執行我的提案：
為了解決上述問題，我們需要重新梳理玩家進入《靈相虛界》的完整流程，並對大廳介面進行徹底的視覺與架構升級。

---

## 二、進入遊戲的完整流程設計

玩家從主應用進入《靈相虛界》時，必須遵循以下流程，確保世界觀的沉浸感與邏輯性：

### 1. 首次進入（未創建角色）
- **攔截跳轉**：若玩家尚未創建遊戲角色，點擊「進入遊戲」後，不應直接進入大廳，而是跳轉至「角色創建與命理綁定」頁面。
- **命理綁定**：系統自動讀取玩家在主應用的八字資料，計算出「本命五行」。
- **初始立繪生成**：根據玩家的性別（Male/Female）與本命五行，自動掛載對應的初始立繪（TASK-008 產出的真實立繪）。
- **確認創建**：玩家確認後，正式進入遊戲大廳。

### 2. 日常進入（已創建角色）
- **登入動畫**：顯示簡短的五行流轉動畫（約 1.5 秒），帶出《靈相虛界》Logo。
- **進入大廳**：直接進入重新設計後的大廳介面。

## 三、大廳介面 (Game Lobby) 重設計規範

大廳不應該只是一個「按鈕集合頁」，而應該是玩家在《靈相虛界》中的「個人樞紐」。

### 1. 頂部區域：個人資訊與資源 (Header)
- **左側**：玩家暱稱、等級（若有）、本命五行標籤（需帶有對應的五行顏色，例如木為綠色、火為紅色）。
- **右側**：資源欄位，清晰顯示「天命幣」與「靈石」的數量，點擊旁邊的 `+` 號可直接跳轉至商城儲值/兌換區。
- **背景**：取消目前單調的深色漸層，改用帶有五行流轉光暈的動態背景，或是根據玩家本命五行顯示專屬的背景氛圍。

### 2. 核心視覺區：角色展示 (Hero Section)
- **佔比**：佔據畫面上半部約 40% 的空間。
- **內容**：
  - **真實立繪**：必須正確讀取玩家性別，並掛載 TASK-008 產出的高解析度立繪圖片，絕對不能再用預設的像素小圖。
  - **五行特效**：立繪背後需有對應本命五行的光暈特效。
  - **互動**：點擊角色立繪，可觸發簡單的語音或文字氣泡（例如：「今日流日為水，宜向北行。」）。

### 3. 中部區域：靈相空間與日常 (Personal Space)
這是玩家「先完成個人配置」的核心區域。
- **靈相空間入口**：設計為一個大型的卡片按鈕，背景帶有衣櫃或房間的意象。
  - **標題**：靈相空間
  - **副標題**：換裝・氣場・每日穿搭
  - **狀態提示**：若今日尚未完成穿搭任務，右上角需有紅點或「今日任務未完成」的醒目標籤。
- **今日任務捷徑**：在靈相空間入口下方，獨立拉出一條捷徑，顯示當前進行中的任務（如：穿戴 2 件土屬性服裝），並附帶「前往」按鈕。

### 4. 底部區域：虛相探索與擴展 (World Exploration)
這是玩家配置完畢後，前往「探險」的區域。
- **虛相探索入口**：設計為另一個大型卡片按鈕，背景帶有大地圖或戰鬥的意象。
  - **標題**：虛相探索
  - **副標題**：大地圖・戰鬥・採集
  - **狀態**：若目前處於測試中，需標示「測試中」，但點擊後應能進入測試版地圖或戰鬥介面。
- **天命商城入口**：設計為中型卡片，放置於虛相探索旁邊或下方。
  - **標題**：天命商城
  - **副標題**：靈石・服裝・道具
  - **狀態**：若有新品上架，需有「新品上架」的標籤。

## 四、給主系統的具體實作指令

請主系統開發者依照以下指令進行修改：

1. **修正性別與立繪判斷**：
   - 檢查 `GameLobby.tsx` 中的性別判斷邏輯，確保 `user.gender` 正確對應到 `male` 或 `female`。
   - 將立繪圖片路徑指向 S3 上 TASK-008 產出的真實圖片，替換掉目前的佔位圖。
2. **重構大廳佈局**：
   - 廢棄目前的單欄式列表佈局。
   - 實作「頂部資源欄 + 核心角色展示區 + 靈相空間卡片 + 虛相探索卡片」的層次化佈局。
3. **實作攔截與創建流程**：
   - 在進入 `/game` 路由前，檢查 `game_users` 表中是否有該玩家資料。
   - 若無，強制導向 `/game/create` 頁面進行本命五行計算與初始立繪確認。
4. **視覺細節優化**：
   - 移除畫面上多餘的「天命共振」文字，統一使用「靈相虛界」品牌識別。
   - 確保所有按鈕與卡片的圓角、陰影符合現代手遊的 UI 規範。
