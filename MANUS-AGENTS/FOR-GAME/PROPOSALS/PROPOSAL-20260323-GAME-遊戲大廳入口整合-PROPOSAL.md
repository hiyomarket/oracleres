## 遊戲功能說明 [必填]

### 功能名稱
遊戲大廳首頁與主導覽列整合 (Game Lobby & Navigation Integration)

### 功能描述
先提出我的看法：目前我們已經做好了 `/game/avatar`（靈相空間）、`/game/shop`（商城）和 `/game/combat`（戰鬥測試），但這些頁面像是一座座孤島，用戶在主系統裡找不到入口進去。

再來執行你的提案：本提案將建立一個統一的「遊戲大廳（`/game`）」，作為所有遊戲化模組的樞紐。同時，我們會在主系統的 `SharedNav`（導覽列）中加入「天命共振」的入口，讓用戶能從任何地方一鍵進入遊戲世界。

這是一個**高衝擊目標**，它標誌著遊戲化模組正式與主系統「合體」，讓用戶體驗無縫銜接。

### 用戶遊玩流程
1. 用戶登入主系統後，在頂部導覽列看到新增的「天命共振 (Game)」按鈕。
2. 點擊後進入 `/game`（遊戲大廳首頁）。
3. 大廳首頁會顯示用戶當前的「靈相角色預覽（縮圖）」、「靈石餘額」、「天命幣餘額」。
4. 畫面中央提供三個主要入口卡片：
   - **靈相空間**（前往 `/game/avatar`，可看到「今日任務未完成」的紅點提示）
   - **天命商城**（前往 `/game/shop`）
   - **虛相探索**（前往 `/game/combat`，標註「測試中」）
