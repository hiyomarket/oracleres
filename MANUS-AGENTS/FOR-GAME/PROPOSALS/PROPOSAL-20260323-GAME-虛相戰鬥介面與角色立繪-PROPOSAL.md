## 遊戲功能說明 [必填]

### 功能名稱
虛相戰鬥介面與角色立繪整合 (Virtual World Combat UI & Player Avatar Integration)

### 功能描述
先提出我的看法：目前靈相世界的換裝系統已經上線，但虛相世界（打怪/採集）還是一片空白。為了讓戰鬥系統能順利開發，我們需要先建立一個「戰鬥介面框架」，並把玩家的角色放進去，確保動作切換順暢。

再來執行你的提案：本提案將實作虛相世界的基礎戰鬥介面（`/game/combat`），並整合 TASK-008 的 6 張玩家角色立繪（男女各 3 種狀態：待機、攻擊、受傷）。此階段**不包含**真實的戰鬥數值計算與怪物 AI，僅實作「前端介面佈局」與「角色狀態切換測試」。

這是一個**高衝擊目標**，能讓團隊立刻看到虛相世界的視覺雛形，為後續的數值與技能系統打下基礎。

### 用戶遊玩流程（測試版）
1. 用戶進入 `/game/combat` 測試路由。
2. 畫面右側顯示玩家角色（依據用戶性別自動載入對應的立繪），預設為「待機 (Idle)」狀態。
3. 畫面下方顯示測試用的操作按鈕：「攻擊測試」、「受傷測試」。
4. 點擊「攻擊測試」，角色切換為攻擊立繪，0.5 秒後切回待機。
5. 點擊「受傷測試」，角色切換為受傷立繪，0.5 秒後切回待機。
