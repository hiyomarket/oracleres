## 遊戲功能說明 [必填]

### 功能名稱
每日穿搭任務與 Aura Score 獎勵機制 (Daily Outfit Quests & Aura Score Rewards)

### 功能描述
先提出我的看法：目前用戶一進來就會獲得一套符合本命五行的初始服裝，但如果沒有「動機」，他們可能換完一次就不再打開衣櫃了。我們需要一個機制，讓「每天換衣服」變成一種有回饋的習慣。

再來執行你的提案：本提案將實作「每日穿搭任務系統」。系統會根據當日的「環境五行運勢」（例如今日火旺），隨機生成一個穿搭挑戰（例如：穿戴至少兩件水屬性服裝來平衡火氣）。用戶完成挑戰後，提交當日穿搭，即可獲得額外的 Aura Score 加成與少量「靈石」獎勵。

這是一個**高衝擊目標**，能有效提升用戶的每日回訪率，並為未來的「虛擬服裝商城」創造靈石的消耗與獲取循環。

### 命理連結說明 [必填]
任務的生成邏輯直接與「每日超級流日報告」中的五行運勢連動。例如，若流日報告建議用戶今日需補「金」，穿搭任務就會要求用戶穿戴金屬性服裝。這讓遊戲化體驗成為命理建議的「實踐場」。

### 用戶遊玩流程
1. 用戶進入靈相空間（`/game/avatar`）。
2. 畫面上方顯示今日任務卡片：「今日火氣過旺，請穿戴至少 2 件【水屬性】服裝來降溫。」
3. 用戶打開衣櫃，換上水屬性服裝。
4. 點擊「提交今日穿搭」按鈕。
5. 系統結算 Aura Score，若符合任務條件，觸發「任務完成」動畫，額外發放 +20 Aura Score 與 50 顆靈石。

## UI 設計說明 [必填]

### 版面配置
在 `AvatarRoom.tsx` 的右側或上方，新增一個懸浮的任務卡片：

```text
┌─────────────────────────────────────────┐
│  [任務卡片]                             │
│  🌟 今日挑戰：水之調和                  │
│  說明：今日火旺，請穿戴 2 件水屬性服裝。│
│  進度：1/2 (未完成)                     │
│  獎勵：+20 Aura, 💎 50 靈石             │
└─────────────────────────────────────────┘
```

### 視覺風格
- **重點配色優化**：任務卡片的邊框與進度條，應根據任務要求的五行屬性動態變色（例如水屬性任務使用 `#00CED1` 量子青）。
