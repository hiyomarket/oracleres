## 遊戲功能說明 [必填]

### 功能名稱
遊戲大廳角色立繪升級

### 功能描述
先提出我的看法：目前遊戲大廳 (`GameLobby.tsx`) 的角色縮圖只是五行色塊，雖然能表達屬性，但缺乏「這是一個遊戲」的代入感。既然美術 Agent 已經完成了 TASK-008 的虛相世界玩家立繪，我們應該立刻把它放上首頁。

再來執行你的提案：本提案將修改 `GameLobby.tsx`，根據用戶的性別，載入對應的 `player_male_idle.png` 或 `player_female_idle.png`。這是一個**高衝擊目標**，能用極低的開發成本，大幅提升用戶進入遊戲大廳的第一印象。

### 實作細節
1. **取得用戶性別**：透過現有的 `useUser()` hook 或 `getUserProfileForEngine` 取得用戶設定的性別。
2. **圖片路徑對映**：
   - 男生：使用 `ART/OUTPUTS/TASK-008/player_male_idle.png`（或對應的 S3 CDN 網址）
   - 女生：使用 `ART/OUTPUTS/TASK-008/player_female_idle.png`（或對應的 S3 CDN 網址）
3. **樣式調整**：
   - 移除原有的五行色塊背景。
   - 將立繪圖片設定為 `object-contain`，並確保在不同螢幕尺寸下都能完整顯示。
   - 保留原有的五行光暈特效（例如木屬性玩家，立繪背後仍有 `#2E8B57` 的微光）。
