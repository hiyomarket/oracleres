# 魔力寶貝（Cross Gate）戰鬥系統研究報告

**研究目的**：為天命共振遊戲戰鬥系統提供數值設計參考
**資料來源**：魔力百科 molibaike.com、巴哈姆特 wiki2.gamer.com.tw、17173.com
**研究日期**：2026-03-24
**負責人**：大管家

## 一、屬性系統（4屬性）

### 1.1 四屬性相剋表

| 屬性 | 克制 | 被克制 |
| :--- | :--- | :--- |
| **地** | 水 | 風 |
| **水** | 火 | 地 |
| **火** | 風 | 水 |
| **風** | 地 | 火 |

### 1.2 屬性修正值（核心數值）

| 剋制程度 | 修正值 | 對敵傷害 | 受敵傷害 |
| :--- | :--- | :--- | :--- |
| **全剋** | 0.3 | **+30%** | **-20%** |
| **半剋** | 0.15 | +15% | -10% |
| 無關屬性 | 0 | 正常 | 正常 |

> ⚠️ 重要：修正值為 **0.3**（非多數人以為的 0.2），這是物理與魔法傷害通用的核心數值。

### 1.3 屬性相剋計算公式（進階）

**基礎公式**：
```
修正值 = 0.3 × (自屬性A × 敵被剋屬性 + 自屬性B × 敵被剋屬性...)
        - 0.3 × (自屬性A × 敵剋制屬性 + 自屬性B × 敵剋制屬性...)
```

**計算要點**：
1. 攻守雙方位置固定，不能隨意調換
2. 每種屬性只能參與一次計算，不能重複
3. 最終修正值直接加進傷害倍率

**範例**：
- 5地5水 攻 5風5地 → 修正值 -0.15（半剋效果，傷害 -15%）
- 7風3地 攻 4火6風 → 修正值 -0.138（約 -14%，接近半剋）

### 1.4 特殊屬性類型

| 類型 | 說明 | 實例 |
| :--- | :--- | :--- |
| **相生屬性** | 最常見，屬性間有相生相剋 | 5風5地、6火4風 |
| **無關屬性** | 既不相生也不相剋，純防禦型 | 5風5水（相互抵消） |
| **全屬性** | 每種屬性都有，極稀有 | 露比（全3）、完美黃金幼龍（全7） |
| **大屬性** | 屬性合計 > 10，強度極高 | 10地10水（合計20） |

> 💡 無關屬性是**被動防禦屬性**：別人很難克制它，但它也無法克制別人。

## 三、物理傷害公式

### 3.1 核心機制：攻防 240 理論

當攻擊或防禦 ≤ 240 時：
- 實際攻擊力 = 面板攻擊力
- 實際防禦力 = 面板防禦力

當攻擊或防禦 > 240 時（遞減報酬）：
- 實際攻擊力 = 240 + 30% × (攻擊力 - 240)
- 實際防禦力 = 240 + 30% × (防禦力 - 240)

> 💡 這個 240 分界點（部分私服為 241 或 250）是物理傷害遞減報酬的關鍵閾值。

### 3.2 基礎傷害公式

```
最終攻擊 = [ (技能攻擊加成/100) × 面板攻擊 + 面板攻擊 ]
最終防禦 = [ (技能防禦加成/100) × 面板防禦 + 面板防禦 ]

非必殺傷害 = ATK² / (ATK/3 + DEF)

必殺額外傷害 = [ 敵方面板防禦 × ATK等级 / 敵方等级 / 2 ]
必殺傷害 = 必殺額外傷害 + 非必殺傷害
```

### 3.3 最終物理傷害計算

```
物理傷害 = 非必殺傷害（或必殺傷害）
         × (100 + 屬性剋制 + 種族剋制)% / 100
         × rand(90, 110)%（傷害波動）
         × [石化折扣率 × 野外系數 × 武器傷害倍率...]
         × (100 - 祈禱傷害減傷)% / 100

最終傷害 = [ 物理傷害 × (1 - 聖盾折扣率) ] × 制御巫術倍率
```

**最終傷害最低為 1**。

### 3.4 武器類型傷害倍率

| 武器類型 | 倍率說明 |
| :--- | :--- |
| 小刀 | rand(5,8) × 0.1（0.5~0.8） |
| 回力 | rand(1,6) × 0.1（0.1~0.6） |
| 氣功彈（格鬥） | 無武器倍率 |

## 四、魔法傷害公式

### 4.1 精神（MND）比例係數 A

精神比例 = (自身精神 / 對方精神) × 100

| 精神比例 | 角色攻擊係數A | 寵物攻擊係數A |
| :--- | :--- | :--- |
| > 119% | 1.1 | 1.1 |
| > 113% | 1.0 | 1.1 |
| > 105% | 0.9 | 1.05 |
| > 97% | 0.7 | 0.95 |
| > 89% | 0.6 | 0.9 |
| > 79% | 0.4 | 0.7 |
| > 69% | 0.3 | 0.6 |
| ≤ 69% | **0.1** | **0.4** |

> ⚠️ 精神比例對魔法傷害影響極大，低於 69% 時係數只剩 0.1！

### 4.2 係數 B（精神滿足度）

```
係數B = 0.8 × (自身精神 / 所需精神)    [範圍 0 ~ 0.8]
```

- 精神越接近所需門檻，係數B越高
- 達到門檻時係數B = 0.8（最高）
- 精神低於門檻，係數線性下降

### 4.3 係數 C（魔攻滿足度）

```
係數C = (自身魔攻 - 等級所需魔攻) / 等級 / 10    [範圍 0 ~ 0.4]
```

- 寵物魔攻為 0（故寵物魔法傷害主要靠精神撐）
- 達到門檻時係數C = 0.4（最高）

### 4.4 魔法傷害基本公式

```
傷害A = 基本值 × 係數A × (係數B + 係數C)

魔法屬性加成（精靈卵變身後）：
  IF 自身屬性 = 地 → 地屬性 × [RAND(95,105)%]
  IF 自身屬性 = 水 → 水屬性 × [RAND(95,105)%]
  ...（其餘屬性同）
  同時：火魔法被地屬性加成（剋制關係）

最終傷害 = 傷害A
          × RAND(95,105)%
          × (100 + 種族剋制 + 係數D)%
          × (100 + 係數E)%   [係數E = 屬性加成]
```

### 4.5 魔法基本值（單體魔法，等級 01-10）

| 等級 | 單魔 | 強魔 | 超魔 | 吸血 |
| :--- | :--- | :--- | :--- | :--- |
| 01 | 95 | 57 | 38 | 63 |
| 05 | 368 | 221 | 147 | 245 |
| 10 | 722 | 433 | 289 | 481 |

## 五、抗魔減傷系統

### 5.1 抗魔減傷表

抗魔減傷率由「抗魔值」與「魔法等級」共同決定，採用**區間制**：

- 抗魔區間 = [X×16+1, (X+1)×16]，X 為抗魔值所在的級別（0~19）
- 抗魔 280 = 區間 [273, 288] = 等級 17
- 減傷率由萬分之一隨機表決定（180/450/930/1510/3860/1510/930/450/180 萬分比對應9種減傷率：1/2/3/5/9/13/17/21/25%）

### 5.2 抗魔減傷率層級

| 減傷率 | 說明 |
| :--- | :--- |
| 1% | 微減傷 |
| 5% | 輕微 |
| 9% | 中等 |
| 13% | 顯著 |
| 17% | 高等 |
| 21% | 極高 |
| 25% | 封頂 |

> 💡 抗魔系統的**隨機性**是魔力寶貝魔法戰鬥的核心特色——同樣抗魔值，每次被魔法命中有不同減傷結果。

## 六、對天命共振遊戲的設計建議

### 6.1 與現有設計的對照

| 天命共振（五行） | 魔力寶貝（四屬性+種族） | 借鑒點 |
| :--- | :--- | :--- |
| 木/火/土/金/水 | 地/水/火/風 | 五行比四屬性多一環，相剋邏輯可更複雜 |
| 屬性抗性（GD-002） | 抗魔減傷系統 | 可引入**隨機減傷區間**機制增加趣味 |
| 怪物圖鑑（GD-011A~E）| 種族系統（10種） | 可參考多鏈剋制關係設計怪物族系 |
| 戰鬥AI（GD-005） | 等級一~五AI | 現有設計已相當完整 |

### 6.2 建議引入的參考機制

1. **屬性修正 0.3 制**：目前天命共振的 +25% 剋制可考慮調整為 +30%（更接近魔力寶貝的數值深度）
2. **抗魔隨機減傷**：可增加「抗魔值」維度，每次魔法攻擊有隨機減傷區間
3. **屬性水晶系統**：參考「風地水火」屬性，讓怪物與角色都有**複合屬性**（如 7風3地）
4. **種族剋制**：可在怪物圖鑑中新增「種族」維度，增加策略深度

### 6.3 數值平衡參考

| 魔力寶貝 | 天命共振（目前） | 建議調整 |
| :--- | :--- | :--- |
| 全剋 +30%傷 | 五行剋制 +25%傷 | 建議提升至 +30% |
| 半剋 +15%傷 | - | 建議新增半剋層次 |
| 防禦遞減報酬（240閾值） | 減傷率 = DEF/(DEF+100) | 方向一致，閾值可微調 |
| 暴擊 = 等級 ATK²/DVF | 火屬性/暴擊率 | 邏輯相似，可繼續沿用 |
