## 一、屬性系統（四屬性）

### 1.1 四屬性相剋規則

| 屬性 | 克制（攻擊加成） | 被克制（受傷加成） |
| :--- | :--- | :--- |
| **地** | 水（+30%） | 風（受傷+20%） |
| **水** | 火（+30%） | 地（受傷+20%） |
| **火** | 風（+30%） | 水（受傷+20%） |
| **風** | 地（+30%） | 火（受傷+20%） |

> 核心規則：地→水→火→風→地，構成完整循環。每一個屬性只剋一個、被一個剋。

---

### 1.2 屬性水晶的種類與結構

#### 普通水晶（10格屬性）

| 水晶類型 | 屬性組合 | 合計格數 |
| :--- | :--- | :--- |
| 純水晶 | 10地 / 10水 / 10火 / 10風 | 10 |
| 相剋水晶 | 5地5水 / 6火4風 等 | 10 |
| 無關水晶 | 5風5水 / 5地5火 | 10 |

> ⚠️ 風水、地火 為「無關屬性」：既不相剋也不相生，是純防禦型。兩者對戰時加成完全抵消（攻防皆為 0）。

#### 特殊水晶（6格屬性，多出2格算全剋）

| 水晶類型 | 說明 |
| :--- | :--- |
| 希望水晶 | 相生組合，多出2格視同全剋額外加成 |
| 強襲水晶 | 攻擊型 |
| 強壓水晶 | 防禦型 |

---

### 1.3 屬性修正係數（全剋 vs 半剋）

| 剋制程度 | 對敵傷害 | 受敵傷害 | 係數值 |
| :--- | :--- | :--- | :--- |
| **全剋** | +30% | -20% | **0.3** |
| **半剋** | +15% | -10% | 0.15 |
| **無剋制** | ±0% | ±0% | 0 |
| **被半剋** | -15% | +10% | -0.15 |
| **被全剋** | -30% | +20% | -0.3 |

> ⚠️ 最重要發現：修正係數是 **0.3**（不是多數人以為的 0.2！）

---

### 1.4 屬性修正完整公式

單一屬性水晶（合計10格）公式：
```
屬性加成% = 100 + 30 × [(我方屬性 × 敵方被剋屬性) - (我方屬性 × 敵方剋制屬性)]
```

#### 實戰範例

**範例一：5風5地 vs 5風5地（互為半剋）**
```
修正值 = 0.3×(5風×5地) - 0.3×(5風×5地) = 0 → 完全無加成
```

**範例二：10火 vs 10地（全剋）**
```
修正值 = 0.3×(10火×0) + ... = +30% 傷 / -20% 受傷
```

**範例三：全3屬性（各3，共12格）vs 5風5地**
```
全屬性：先抵消等量屬性（3風抵5地→剩2風；3地抵5風→剩2地）
剩餘：2風+3火 vs 2地+3水
修正 ≈ 0 → 全3對雙5屬性完全抵消（防禦型）
```

## 三、物理傷害公式

### 3.1 攻防 240 理論（核心閾值）

| 攻/防數值 | 實際效應值計算 |
| :--- | :--- |
| ≤ 240 | 直接等於面板數值 |
| > 240 | 240 + 30% × (面板數值 - 240) |

實例：
| 面板數值 | 實際效應值 |
| :--- | :--- |
| 100 | 100 |
| 200 | 200 |
| 240 | 240 |
| 300 | 258 |
| 400 | 288 |
| 500 | 318 |
| 1000 | 468 |

> 💡 當防禦超過 240 後，投入防禦的邊際效益急劇下降（每增加100點面板防禦，只增加30點實際防禦）。這個「240閾值」是物理職業成長的關鍵瓶頸。

---

### 3.2 中值傷害公式（無必殺、無屬性、無種族、無武器）

```
標準普通攻擊中值傷害 = [ 3 × (實際攻擊力)² ] / (實際攻擊力 + 實際防禦力 × 3)
```

等價變形：
```
= 實際攻擊力 / (1/3 + 實際防禦力 / 實際攻擊力)
```

---

### 3.3 傷害波動系統（90%~110%）

每次物理攻擊，系統隨機抽取 21 種傷害倍率：
90%, 91%, 92%, 93%, 94%, 95%, 96%, 97%, 98%, 99%, **100%**, 101%, 102%, 103%, 104%, 105%, 106%, 107%, 108%, 109%, 110%

**中值（100%）**為基準，高低各 10% 浮動。

---

### 3.4 武器銳利度修正

| 武器狀態 | 效果 |
| :--- | :--- |
| 攜帶武器（任何） | 傷害 ×1.15（+15%）（格鬥士自帶）|
| 小刀額外 | 50%~80% 隨機浮動 |
| 回力標額外 | 10%~60% 隨機浮動 |
| 氣功彈（格鬥） | 無武器倍率加成 |

---

### 3.5 必殺（暴擊）系統

**必殺額外傷害公式**：
```
必殺額外傷害 = floor(敵方面板防禦 × 攻方等級 / 敵方等級 / 2)
必殺總傷害 = 必殺額外傷害 + 普通傷害
```

> 📝 巴哈姆特版本：`必殺時損傷 = [敵方防禦力 × 攻方等級 / (敵方等級 × 2)] + 原損傷值`
> 兩者本質相同（取整方式略有差異）

---

### 3.6 完整物理傷害計算流程

```
步驟1：面板攻擊 → 實際攻擊力（240閾值遞減）
步驟2：面板防禦 → 實際防禦力（240閾值遞減）
步驟3：中值傷害 = 3×ATK² / (ATK + DEF×3)
步驟4：套用傷波 = 中值 × rand(90%~110%)
步驟5：武器加成 = ×1.15（如有攜帶武器）
步驟6：屬性加成 = ×(1 + 屬性修正)（如火打風+30%）
步驟7：種族加成 = ×(1 + 種族修正)（如人打野獸+20%）
步驟8：必殺加成 = + floor(敵DEF×我LV/敵LV/2)（如有必殺）
步驟9：技能倍率 = ×技能倍率（如崩擊等）
最終傷害 = max(普通傷害, 必殺總傷害)
```

## 四、魔法傷害公式

### 4.1 精神（MND）的核心地位

精神是魔法傷害的**第一決定要素**，人物與寵物有完全不同的計算邏輯。

#### 人物 vs 寵物的精神應用差異

| 項目 | 人物魔法 | 寵物魔法 |
| :--- | :--- | :--- |
| 精神門檻 | 有嚴格門檻，低於門檻傷害大幅下降 | 無嚴格門檻，魔法等級越高傷害越高 |
| 精神影響方式 | 跳躍式 | 平滑上升 |
| 寵物魔攻 | — | 固定為 0（完全靠精神撐） |
| 多餘精神 | **無用** | 對魔時有保護作用 |

#### 人物魔法精神門檻表（巴哈姆特版本）

| 魔法等級 | 所需最低精神 | 單體 | 強力 | 超強 |
|:---:|:---:|:---:|:---:|:---:|
| 1 | 123 | 85 | 50 | 35 |
| 2 | 143 | 150 | 90 | 60 |
| 3 | 163 | 210 | 125 | 85 |
| 4 | 183 | 270 | 160 | 105 |
| 5 | 203 | 325 | 195 | 130 |
| 6 | 223 | 380 | 230 | 150 |
| 7 | 243 | 440 | 260 | 175 |
| 8 | 263 | 500 | 300 | 200 |
| 9 | 283 | 570 | 340 | 230 |
| 10 | **303** | **635** | **380** | **255** |

> 💡 **303精神**是法師的標誌性門檻：達到303精神才能發揮10級魔法完整傷害。

---

### 4.2 精神比例係數 A

精神比例 = (自身精神 / 敵方精神) × 100

| 精神比例 | 角色攻擊係數 | 寵物攻擊係數 |
|:---:|:---:|:---:|
| > 119% | **1.1** | **1.1** |
| 113% ~ 119% | 1.0 | 1.1 |
| 105% ~ 113% | 0.9 | 1.05 |
| 97% ~ 105% | 0.7 | 0.95 |
| 89% ~ 97% | 0.6 | 0.9 |
| 79% ~ 89% | 0.4 | 0.7 |
| 69% ~ 79% | 0.3 | 0.6 |
| ≤ 69% | **0.1** | **0.4** |

> ⚠️ 當精神比例低於 69%，法師的魔法係數只剩 **0.1**！精神低的法師打高精神法師「打不動」。

---

### 4.3 係數 B 與係數 C

```
係數B = 0.8 × (自身精神 / 等級所需精神)    [範圍：0 ~ 0.8]
係數C = (自身魔攻 - 等級所需魔攻) / 等級 / 10    [範圍：0 ~ 0.4]
```

等級所需魔攻：`10 + 30 × (等級 - 1)`
- 等級1：10；等級5：130；等級10：**280**

---

### 4.4 完整魔法傷害公式

```
傷害A = 基本值 × 係數A × (係數B + 係數C)

最終魔法傷害 =
  傷害A
  × RAND(95%~105)%                    （傷波）
  × (100 + 種族剋制 + 係數D)%         （種族+屬性加成）
  × (100 + 係數E)%                    （係數E=屬性加成）
  × 野外加成 × 祈禱削弱 × 抗魔削弱 × 石化削弱 × 跳舞加成 × 怪物時段加成
```

> ⚠️ 道具服版本：人物魔法傷害約 ×1.45~1.5；寵物魔法傷害約 ×2.0

## 五、抗魔減傷隨機系統

這是魔力寶貝最獨特的設計——抗魔減傷並非固定值，而是由「抗魔值」與「魔法等級」共同決定隨機區間。

### 5.1 抗魔減傷概率萬分表

縱軸：進攻方魔法等級（1~10）
橫軸：防守方抗魔值所屬級別（0~19）

```
萬分位（由左至右）：
180  |  450  |  930  |  1510  |  3860  |  1510  |  930  |  450  |  180
 1%  |   2%  |   3%  |   5%   |   9%   |   13%  |  17%  |  21%  |  25%
```

### 5.2 抗魔級別區間

| 抗魔值 | 所屬級別 | 抗魔區間 |
|:---:|:---:|:---|
| 0 | 0 | [1, 16] |
| 1 | 1 | [17, 32] |
| 17 | 17 | [273, 288] |
| 19 | 19 | [305, 999999] |

> 級別公式：X = floor(抗魔值 / 16)，抗魔區間 = [X×16+1, (X+1)×16]

### 5.3 實例說明

情境：敵人抗魔 280（級別17，區間[273,288]），我使用10級魔法
```
1. 抗魔280 → 級別17 → 區間[273,288] ✓
2. 魔法等級10 → 萬分表第10列
3. 隨機抽萬分位（例：抽中3860）
4. 對應減傷率 = 9%
最終魔法傷害 = 原始傷害 × 91%
```

### 5.4 減傷率期望值分析

| 減傷率 | 機率 | 說明 |
|:---:|:---:|:---|
| 1% | 1.8% | 極輕微 |
| 5% | 15.1% | 中度 |
| **9%** | **38.6%** | **最常見（峰值）** |
| 13% | 15.1% | 顯著 |
| 25% | 1.8% | 封頂 |

> 💡 平均期望值約 7~8% 減傷。抗魔值只影響「減傷區間分佈」，不直接等比轉化為減傷率。

```
合擊總傷害 = Σ(各攻擊者傷害) × (1 + 合擊人數 × 10%)
```

|:---:|:---|
| 2人 | ×1.2 |
| 3人 | ×1.3 |
| 4人 | ×1.4 |

## 八、對天命共振的具體建議

### 8.1 與現有設計的完整對照

| 天命共振設定 | 魔力寶貝機制 | 建議動作 |
| :--- | :--- | :--- |
| 五行：木/火/土/金/水 | 四屬性：地/水/火/風 | 五行比四屬多一環，可設計木→火→金→水→木額外迴路 |
