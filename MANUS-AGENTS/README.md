# MANUS-AGENTS — 入口索引

> 這是 Manus Agent 的專屬閱讀區
> **請每次先讀這個檔案**

---

## 資料夾結構

| 資料夾 | 誰來讀 |
|--------|--------|
| `FOR-SYSTEM/` | Manus System Agent（系統建造） |
| `FOR-ART/` | Manus Art Agent（美術設計） |

---

## 每次工作前，請先讀這裡！

### Manus System Agent
1. 先讀 `FOR-SYSTEM/CURRENT-TASK.md`
2. 再讀 `INSTRUCTIONS/` 裡的詳細指令

### Manus Art Agent
1. 先讀 `FOR-ART/CURRENT-TASK.md`
2. 再讀 `SPECS/` 裡的詳細規格

---

## 產出上傳位置

| 產出 | 上傳到 |
|------|--------|
| 美術素材 | `ART/OUTPUTS/` 對應資料夾 |
| 素材清單 | `ART/MANIFEST.json` |
| 任務狀態 | `STATUS/對應任務/STATUS.md` |

---

## 重要說明

- 網站程式碼存放於 **Manus System Agent 本地**，不需要上傳到 GitHub
- GitHub 的 `ART/OUTPUTS/` 是用來讓 System Agent 抓取美術素材用的
- `CLAW-TEAM/` 資料夾是 ClawTeam 內部文件，**不需要讀取**
