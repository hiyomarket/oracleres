# TASK-006：虛相世界 - 五行魔物立繪生成

**發送者**：遊戲 Agent
**接收者**：美術 Agent
**日期**：2026-03-23
**優先級**：🟡 中（為後續戰鬥系統與寵物系統預作準備）

---

## 任務背景

先提出我的看法：目前靈相世界（紙娃娃換裝）的素材已經非常豐富，但我們即將進入「虛相世界」的開發。虛相世界的核心是「戰鬥與捕捉」，我們需要一批具有五行屬性的「魔物（可捕捉成為寵物）」立繪，讓玩家在地圖探索時有目標可以挑戰。

再來執行你的提案：請美術 Agent 提前開始生成虛相世界的「五行魔物立繪」。這些魔物將出現在橫向回合制戰鬥的畫面上，並在被捕捉後成為玩家的寵物。

---

## 任務需求

請設計並生成 **5 隻基礎魔物**，分別對應金、木、水、火、土五行屬性。

### 1. 設計風格與原則
- **風格**：日系奇幻 RPG 風格（類似《魔力寶貝》或《原神》的怪物設計），帶有輕微的賽博龐克或現代元素以符合「天命共振」的世界觀。
- **定位**：這些是「初階魔物」，外觀應該**可愛但帶有一點攻擊性**，讓玩家有想捕捉來當寵物的慾望。
- **視角**：**右側 45° 視角**（因為在橫向戰鬥中，怪物通常站在畫面左側，面向右側的玩家角色）。

### 2. 五行魔物具體設定

| 屬性 | 魔物名稱（暫定） | 設計關鍵字 | 重點配色 |
| :--- | :--- | :--- | :--- |
| **木** | 葉刃狐 | 狐狸、藤蔓尾巴、靈動、自然 | 翠綠 `#2E8B57` |
| **火** | 燼火犬 | 幼犬、四足帶火、熱情、爆發 | 朱紅 `#DC143C` |
| **土** | 岩甲龜 | 烏龜、背甲如礦石、穩重、防禦 | 琥珀棕 `#CD853F` |
| **金** | 刃羽鳥 | 猛禽幼鳥、金屬羽毛、銳利、速度 | 天命金 `#C9A227` |
| **水** | 淵波水母 | 漂浮水母、水波觸手、神秘、魔法 | 量子青 `#00CED1` |

### 3. 技術規格要求（嚴格遵守）

1. **尺寸**：1024x1024 px。
2. **背景**：必須為**絕對透明背景**（PNG 格式），邊緣需乾淨，不可有殘留的白色或黑色毛邊，以免在戰鬥背景上穿幫。
3. **構圖**：魔物主體應佔據畫面的 70-80%，不要裁切到邊緣。
