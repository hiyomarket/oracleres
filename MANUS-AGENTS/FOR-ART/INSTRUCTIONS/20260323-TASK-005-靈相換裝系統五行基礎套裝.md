## 任務需求

請基於 TASK-003 與 TASK-004 已完成的「男女素體（正面、左側 45°、右側 45°）」，為每個性別繪製 5 套基礎服裝。

### 1. 五行套裝設計方向

每套服裝必須包含：**髮型 (hair)**、**上衣 (top)**、**下衣 (bottom)**、**鞋子 (shoes)**。
（飾品與背景本次暫不需繪製）

| 五行屬性 | 設計關鍵字 | 建議主色調（重點配色） |
| :--- | :--- | :--- |
| **木 (Wood)** | 自然、生機、藤蔓、輕盈 | 青綠色系 |
| **火 (Fire)** | 熱情、火焰、俐落、龐克 | 紅紫色系 |
| **土 (Earth)** | 穩重、大地、布料厚實、工裝 | 黃棕色系 |
| **金 (Metal)** | 銳利、金屬配件、科技感、西裝 | 白金色系 |
| **水 (Water)** | 流動、波浪、絲綢、寬鬆 | 黑藍色系 |

### 2. 技術規格要求（嚴格遵守）

1. **圖層拆分**：每套服裝必須嚴格拆分為 `hair`、`top`、`bottom`、`shoes` 四個獨立的 PNG 檔案。
2. **視角對應**：每個部件都需要繪製「正面」、「左側 45°」、「右側 45°」三個版本，以配合前端的動態轉身效果。
3. **對齊基準**：所有輸出的 PNG 畫布尺寸必須完全一致（例如 1024x1024），且服裝在畫布上的位置必須與「素體」的中心點絕對對齊。前端是採用絕對定位疊加，若未對齊會導致衣服穿歪。
4. **背景透明**：所有 PNG 必須為透明背景。
