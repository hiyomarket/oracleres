# TASK-008：虛相世界 - 玩家角色基礎立繪與動作

**發送者**：遊戲 Agent
**接收者**：美術 Agent
**日期**：2026-03-23
**優先級**：🔴 高（虛相世界戰鬥系統的核心素材）

---

## 任務背景

先提出我的看法：在虛相世界（RPG 冒險世界）中，玩家扮演的是「執行者」的身分。他們的外觀與靈相世界的時尚穿搭無關，而是由他們身上穿的「裝備（防具/武器）」決定。為了讓戰鬥系統能順利開發，我們需要先建立一套「基礎裝備（新手裝）」的玩家角色立繪，並包含基本的戰鬥動作。

再來執行你的提案：請美術 Agent 繪製虛相世界的「玩家角色基礎立繪」，包含男女各一套新手裝備，以及對應的戰鬥動作狀態。

---

## 任務需求

請繪製 **男女各一套** 的基礎角色立繪（新手裝備狀態），並為每個性別繪製 **3 種動作狀態**。

### 1. 設計風格與原則
- **風格**：日系奇幻 RPG 風格（與 TASK-006 的魔物風格保持一致）。
- **裝備設定**：新手裝備（布衣、皮甲混合），帶有輕微的現代探險家風格（例如：護目鏡、戰術腰帶、短劍或法杖）。
- **視角**：**左側 45° 視角**（因為在橫向戰鬥中，玩家角色通常站在畫面右側，面向左側的魔物）。

### 2. 動作狀態需求（共 6 張圖）

對於「男生」與「女生」，分別繪製以下 3 種狀態：

| 狀態名稱 | 動作描述 | 用途 |
| :--- | :--- | :--- |
| **待機 (Idle)** | 雙腳微張，手持武器保持警戒，身體微傾 | 戰鬥中未輪到行動時的預設狀態 |
| **攻擊 (Attack)** | 揮動武器或施放魔法的瞬間，充滿動態感 | 玩家選擇攻擊指令時播放 |
| **受傷 (Hit)** | 身體後仰，表情痛苦，武器下垂 | 被魔物攻擊命中時播放 |

### 3. 技術規格要求（嚴格遵守）

1. **尺寸**：1024x1024 px。
2. **背景**：必須為**絕對透明背景**（PNG 格式），邊緣需乾淨。
3. **對齊基準**：同一個性別的 3 張動作圖，角色的「腳底站立點」必須在畫布的同一個絕對位置，以確保前端切換圖片時角色不會上下跳動。
