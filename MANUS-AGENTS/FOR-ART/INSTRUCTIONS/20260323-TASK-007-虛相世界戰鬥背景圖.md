# TASK-007：虛相世界 - 戰鬥場景背景圖生成

**發送者**：遊戲 Agent
**接收者**：美術 Agent
**日期**：2026-03-23
**優先級**：🟡 中（為後續戰鬥系統預作準備）

---

## 任務背景

先提出我的看法：在橫向回合制戰鬥中，除了角色和魔物，我們還需要對應不同地圖節點的「戰鬥背景圖」。這些背景能讓玩家感受到他們目前身處的環境（例如在公園打卡，戰鬥背景就是森林）。

再來執行你的提案：請美術 Agent 繪製 5 張對應五行地貌的戰鬥背景圖。

---

## 任務需求

請設計並生成 **5 張戰鬥背景圖**，對應金、木、水、火、土五種地貌。

### 1. 設計風格與原則
- **風格**：日系奇幻 RPG 風格，帶有輕微的現代/賽博龐克元素（例如森林中可能隱約可見遠處的霓虹城市天際線）。
- **構圖**：橫向捲軸視角。畫面下方需有平坦的「地面區域」（約佔畫面下方 1/3），供角色與魔物站立；畫面上方為遠景。
- **氛圍**：不要過於搶眼，背景應該是襯托角色與魔物的舞台，色彩對比度可稍微降低。

### 2. 五行地貌具體設定

| 屬性 | 場景名稱 | 設計關鍵字 | 重點配色 |
| :--- | :--- | :--- | :--- |
| **木** | 霓虹神木林 | 巨大樹木、發光藤蔓、遠處城市輪廓 | 翠綠 `#2E8B57` |
| **火** | 熔爐廢墟 | 廢棄工廠、岩漿管線、火星飄散 | 朱紅 `#DC143C` |
| **土** | 晶石礦坑 | 巨大琥珀色水晶、岩洞、微光 | 琥珀棕 `#CD853F` |
| **金** | 數據矩陣 | 賽博空間、金屬幾何結構、流動數據 | 天命金 `#C9A227` |
| **水** | 沉星海岸 | 夜間海灘、發光水母、星空倒影 | 量子青 `#00CED1` |

### 3. 技術規格要求（嚴格遵守）

1. **尺寸**：1920x1080 px（16:9 比例）。
2. **格式**：JPG 或 PNG 皆可，不需透明背景。
3. **無角色**：畫面中**絕對不可**出現任何人物、怪物或 UI 元素，純粹是場景空景。

---

## 交付格式

請將完成的素材放入 `ART/OUTPUTS/TASK-007/` 目錄下，並依照以下命名規則：

```text
bg_battle_[五行].jpg

範例：

```

完成後請更新 `MANIFEST.json`，並通知 Boss 轉達給遊戲 Agent。
