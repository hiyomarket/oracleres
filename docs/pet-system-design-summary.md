# 寵物系統設計摘要

## 核心循環
戰鬥遭遇 → 獸魂甕捕捉 → 獲得寵物(隨機等級+隨機檔位) → 培養(升級→BP成長→屬性提升) → 技能(天生3格+天命3格=6格)

## 捕捉系統
- 捕捉率 = 基礎30% × HP因子 × 等級因子 × 道具加成
- HP因子 = (1 - CurrentHP/MaxHP)^0.8 × 0.7 + 0.3
- 等級因子 = 1 / (1 + 0.02 × (MonsterLv - PlayerLv))
- 道具：獸魂甕(×1.0)、秘銀(×1.15)、星輝(×1.30)、天命(×1.50)
- HP<5%進入重傷狀態，顯示捕捉確認

## BP五維系統
- 體質: HP+8, MP+1, ATK+0.1, DEF+0.1, SPD+0.1
- 力量: HP+2, MP+2, ATK+2.7, DEF+0.3, SPD+0.2
- 防禦: HP+2, MP+2, ATK+0.3, DEF+2.7, SPD+0.2
- 敏捷: HP+3, MP+2, ATK+0.3, DEF+0.3, SPD+2.0
- 魔力: HP+1, MP+10, ATK+0.2, DEF+0.2, SPD+0.1

## 檔位
- S: 200+ BP (0.5%), +30%成長
- A: 150-199 BP (9.5%), +20%成長
- B: 110-149 BP (25%), +10%成長
- C: 70-109 BP (45%), 標準
- D: 30-69 BP (17%), -10%成長
- E: 0-29 BP (3%), -20%成長

## 升級BP成長
- 每級: 3固定BP + 1隨機BP
- 成長型態: 鬥爭(力量+1)/守護(防禦+1)/疾風(敏捷+1)/魔力(魔力+1)/均衡(隨機+1)

## 技能系統
- 天生技能: 3格(捕捉時/Lv20/Lv50解鎖)，種族固定不可刪除
- 天命技能: 3格(Lv15/Lv35/Lv60解鎖)，14種可學，可替換可升級
- 天命技能升級: Lv1-10，累積使用次數(50→3000)

## 種族
- 龍系: HP×1.3
- 不死系: HP×1.2
- 一般系: HP×1.0
- 昆蟲系: HP×0.9
- 植物系: HP×0.8
- 飛行系: HP×1.0

## 屬性剋制(火/風/地/水)
- 剋制: 傷害+20%, 受傷-20%
- 被剋: 傷害-20%, 受傷+20%

## 天命技能14種
物理: 連擊、反擊、崩擊、毒擊
魔法: 火焰魔法、冰凍魔法、風刃魔法、隕石魔法
控制: 昏睡魔法、石化魔法、混亂魔法、中毒魔法、遺忘魔法
輔助: 補血魔法、氣絕回復、攻擊吸收、魔法吸收、明鏡止水

## HP曲線(B檔均衡型)
Lv1:50, Lv10:250, Lv20:500, Lv30:800, Lv50:1400, Lv100:2900

## 三個實例寵物
1. 岩蜥(土/坦克, C檔, 新手)
2. 魔角獸(暗/物攻, B檔, 中期)
3. 暮光龍(龍/全能, S檔, 稀有)
