## 一、人物基礎屬性與能力值系統

### 1.1 五項基礎能力值（加點屬性）

魔力寶貝的人物能力由五項基礎屬性構成，玩家每次升級可獲得加點，分配至以下五項 [^1]：

| 基礎屬性 | 中文名稱 | 主要影響 |
|---------|---------|---------|
| **VIT（Vitality）** | 體力 | 生命值、回復力 |
| **STR（Strength）** | 力量 | 物理攻擊力、生命值 |
| **DEF（Defense）** | 強度 | 物理防禦力、生命值 |
| **AGI（Agility）** | 速度（敏捷） | 敏捷值、行動順序 |
| **INT（Intelligence）** | 魔法 | 魔力值、魔法精神 |

### 1.2 加點對七維屬性的影響

每加一點基礎屬性，對七項戰鬥屬性（七維）的影響如下 [^1]：

| 加點 | 生命 | 魔力 | 攻擊 | 防禦 | 敏捷 | 精神 | 回復 |
|-----|------|------|------|------|------|------|------|
| +1 體力 | **+8** | +1 | +0.1 | +0.1 | +0.2 | -0.3 | **+0.8** |
| +1 力量 | +2 | +2 | **+2** | +0.2 | +0.2 | -0.1 | -0.1 |
| +1 強度 | +3 | +2 | +0.2 | **+2** | +0.2 | +0.2 | -0.1 |
| +1 速度 | +3 | +2 | +0.2 | +0.2 | **+2** | -0.1 | +0.2 |
| +1 魔法 | +1 | **+10** | +0.1 | +0.1 | +0.1 | **+0.8** | -0.3 |

> **設計要點：** 體力主加生命與回復，力量主加攻擊，強度主加防禦，速度主加敏捷，魔法主加魔力與精神。精神值影響魔法傷害的穿透與抵抗。

### 1.3 配點上限（爆點機制）

任何一種能力都不能超過能力總和的一半，此限制稱為「爆點」[^1]。

**公式：** `單項能力上限 = (等級 - 1) × 2 + 15`

例如 Lv.50 的角色，每項能力上限為 `(50-1) × 2 + 15 = 113`，若某項屬性超過此值則無法繼續加點。

### 1.4 七維戰鬥屬性說明

| 屬性名稱 | 功能說明 |
|---------|---------|
| **生命（HP）** | 戰鬥中的血量，歸零即氣絕 |
| **魔力（MP）** | 使用技能的消耗資源 |
| **攻擊** | 物理傷害的基礎值 |
| **防禦** | 減少物理傷害的能力 |
| **敏捷** | 決定行動順序與迴避率 |
| **精神** | 影響魔法傷害與魔法防禦 |
| **回復** | 每回合自然回血量，也影響治療技能效果 |

## 二、傷害結構與計算公式

### 2.1 物理傷害公式

物理傷害的基礎計算公式 [^2]：

```
物理傷害 = (攻擊力 - 防禦力) × 種族修正 × 屬性修正 × 水晶修正 × 技能倍率
```

詳細展開：

```
基礎傷害 = 攻擊力 - 防禦力
最終傷害 = 基礎傷害 × 種族加成(70%~130%) × 水晶屬性加成(70%~130%) × 魔法屬性加成(65%~135%)
```

**傷害顏色分級（白/黃/紫/紅傷）：**

| 傷害顏色 | 計算公式 | 說明 |
|---------|---------|------|
| 白傷 | 等級² × 1 | 基礎傷害 |
| 黃傷 | 等級² × 2.25 | 中等傷害 |
| 紫傷 | 等級² × 4 | 高傷害 |
| 紅傷 | 等級² × 6.25 | 最高傷害 |

### 2.2 魔法傷害公式

魔法傷害主要依賴精神值，公式如下 [^3]：

```
魔法傷害 = (精神力 × 技能倍率) × 屬性修正 × 種族修正
```

精神值（Spirit）的特殊性：
- 精神值越高，魔法攻擊傷害越高
- 精神值也影響魔法防禦（受到魔法傷害時的減免）
- 加點魔法屬性每點增加 +0.8 精神，加點體力每點 **減少** -0.3 精神

### 2.3 治療公式

補血魔法的治療量計算 [^4]：

```
補血量 = 技能等級 × 60 × (回復力%)
強力補血量 = 技能等級 × 36 × (回復力%)
超強補血量 = 技能等級 × 24 × (回復力%)
```

> **說明：** 回復力是七維屬性之一，主要由體力加點提升（+0.8/點）。治療量與回復力直接掛鉤，而非固定數值。

### 2.4 戰鬥總傷害加成公式

```
戰鬥總傷害加成 = 種族加成(70%~130%) × 水晶屬性加成(70%~130%) × 魔法屬性加成(65%~135%)
```

## 三、屬性相剋系統

### 3.1 四大屬性相剋關係

魔力寶貝採用四元素循環相剋 [^5]：

```
地 → 剋 → 水 → 剋 → 火 → 剋 → 風 → 剋 → 地
```

**相剋修正值：** 完全相剋時，傷害修正為 **+0.3（增加 30%）**，而非一般認知的 20%。

### 3.2 混合屬性相剋計算公式

**通用公式：**
```
修正值 = 0.3 × (自地×敵水 + 自水×敵火 + 自火×敵風 + 自風×敵地)
        - 0.3 × (自地×敵風 + 自水×敵地 + 自火×敵水 + 自風×敵火)
```

**計算範例：** 5地5水 攻擊 5風5地：
```
修正 = 0.3 × (-0.5×0.5 - 0.5×0.5) = -0.15
結果：傷害減少 15%（接近半剋效果）
```

### 3.3 特殊屬性類型

| 屬性類型 | 說明 | 特性 |
|---------|------|------|
| **相剋屬性** | 相生相剋的組合（如 5風5地） | 最常見，攻守均衡 |
| **無關屬性** | 既不相生也不相剋（如 5風5水） | 防禦型屬性，難被剋制也難剋制他人 |
| **全屬性** | 四種屬性均有（如全3屬性） | 小全屬性為防禦型，大全屬性為全能型 |
| **大屬性** | 屬性總和超過10（如10地10水） | 進攻型，傷害修正極高 |
| **純屬性** | 單一屬性（如純10地） | 對被剋種族傷害極高，但也易被剋制 |

## 四、種族相剋系統

### 4.1 種族相剋鏈

魔力寶貝共有十種種族，分為兩條相剋鏈 [^5]：

**全剋鏈（增傷 20%）：**
```
人形 → 野獸 → 植物 → 不死 → 人形（循環）
龍 → 飛行 → 昆蟲 → 特殊 → 金屬 → 龍（循環）
```

**半剋鏈（增傷 10%）：**
```
人形 → 昆蟲 → 植物 → 特殊 → 龍 → 野獸 → 飛行 → 不死 → 金屬 → 人形
```

**邪魔系：** 全剋所有種族，但效果特殊：
- 對敵傷害增加 10%
- 受敵傷害減少 20%

### 4.2 種族相剋傷害修正

| 相剋關係 | 對敵傷害 | 受敵傷害 |
|---------|---------|---------|
| **全剋** | +20% | -20% |
| **半剋** | +10% | -10% |
| **被全剋** | -20% | +20% |
| **被半剋** | -10% | +10% |
| **邪魔系** | +10% | -20% |

> **注意：** PK 場所有種族影響減半。轉換種族只影響受到的傷害，不影響打出的傷害。

## 五、人物升級系統

### 5.1 升級經驗公式

```
升至下一等級所需經驗值 = (目標等級)⁴
```

例如升至 10 級需要 10⁴ = 10,000 經驗值。

### 5.2 技能經驗獲取條件

```
技能經驗有效戰鬥條件：怪物等級 > 自身等級 ÷ 2 - 3
```

若怪物等級過低，將無法獲得技能經驗。

### 5.3 加點總量

每升一級獲得一定數量的加點，加點分配至五項基礎能力。非戰鬥系職業在技能 LV5 後，每升一級的能力變化如下 [^1]：

| 職業 | 耐力 | 靈巧 | 智力 |
|-----|------|------|------|
| 護士 | +3 | -2 | +3 |
| 醫生 | +1 | -2 | +5 |
| 採集系 | +5 | +1 | -2 |
| 製造系 | +2 | +4 | -2 |
| 廚師 | +3 | +3 | -2 |
| 藥劑師 | -2 | +2 | +4 |
| 鑑定師 | -2 | +1 | +5 |
| 修理工 | +3 | +3 | -2 |

## 七、寵物 BP 值與屬性分配

### 7.1 BP 值概念

BP（Basic Point，基礎點數）是寵物最核心的成長數值 [^7]。寵物的所有屬性均由 BP 衍生而來。

**寵物五項 BP：**

| BP 名稱 | 說明 |
|---------|------|
| **體力 BP** | 影響生命值、回復力 |
| **力量 BP** | 影響攻擊力 |
| **強度 BP** | 影響防禦力 |
| **敏捷 BP** | 影響敏捷值 |
| **魔法 BP** | 影響魔力值、精神力 |

### 7.2 BP 對七維屬性的影響

每增加一點 BP，對七維屬性的影響如下 [^7]：

| 加點 | 生命 | 魔力 | 攻擊 | 防禦 | 敏捷 | 回復 | 精神 |
|-----|------|------|------|------|------|------|------|
| +1 體力 BP | **+8** | +1 | 0 | 0 | 0 | **+0.8** | 0 |
| +1 力量 BP | +2 | 0 | **+2** | 0 | 0 | 0 | 0 |
| +1 強度 BP | +2 | 0 | 0 | **+2** | 0 | 0 | 0 |
| +1 敏捷 BP | 0 | 0 | 0 | 0 | **+2** | 0 | 0 |
| +1 魔法 BP | 0 | **+4** | 0 | 0 | 0 | 0 | **+1** |

> **注意：** 寵物的 BP 與人物的加點效果略有不同，寵物的力量和強度 BP 效果更為純粹，主要各自對應攻擊與防禦。

### 7.3 寵物 BP 最大值計算

```
野生寵物最高 BP = (星數 × 2) + {(星數 ÷ 2) × (捕獲時等級 - 1)}
```

**星數說明：** 寵物的星數（品質）決定了 BP 的上限，星數越高，成長潛力越大。

### 7.4 爆點機制

當寵物的某一項 BP 高於其他 BP 總數時，該項 BP 無法繼續加點，稱為「爆點」[^7]。

```
爆點條件：某項 BP > 其他所有 BP 之和
```

### 7.5 寵物成長計算

```
寵物成長率 = (目前數值 - 原始數值) ÷ (目前等級 - 原始等級)
```

成長率反映了寵物每升一級平均增加的屬性值，是評估寵物品質的重要指標。

### 7.6 寵物完全聽話等級

```
寵物完全聽話等級 = 主人調教等級 × 12.5
```

若寵物等級超過此值，在戰鬥中可能不服從指令。

## 八、寵物升級與成長系統

### 8.1 寵物種族分類

寵物共有十大種族，與人物的種族相剋系統相同：

人形系、野獸系、植物系、不死系、龍系、飛行系、昆蟲系、特殊系、金屬系、邪魔系

### 8.2 寵物屬性（元素屬性）

寵物擁有元素屬性（地、水、火、風），由水晶決定，影響戰鬥中的屬性相剋計算。

### 8.3 寵物忠誠度

忠誠度影響寵物的服從性與騎乘持續時間：

```
騎乘持續步數 = 基礎步數 × 寵物忠誠度%
```

例如騎乘 LV1 的基礎步數為 100，若忠誠度 80%，則持續 80 步。

## 十二、技能學習與升級方式

### 12.1 人物技能學習方式

人物技能的學習方式主要有三種 [^8]：

**1. NPC 購買：** 前往特定地點的 NPC 購買技能書，支付魔幣費用後學習。

**2. 任務解鎖：** 完成特定任務後才能學習，例如：
- 暗殺：完成「忍者就職」任務
- 毒擊：完成「魔導書抄本（琥珀之卵3）」任務
- 投擲：完成「爆彈師就職」任務

**3. 打倒特定怪物：** 擊敗特定 Boss 或怪物後學習，例如：
- 明鏡止水：打倒改造殭屍後學習
- 超強補血魔法：打倒神獸後找天書學習

### 12.2 技能升級方式

技能升級透過在戰鬥中使用技能來累積技能經驗。每次使用技能可獲得一定的技能經驗值，累積到對應等級所需的經驗值後自動升級。

**技能升級條件：**
1. 必須在有效戰鬥中使用技能（怪物等級需高於自身等級 ÷ 2 - 3）
2. 累積足夠的技能經驗值
3. 部分技能在特定等級後需要額外條件（如任務）

### 12.3 技能格數限制

每個角色擁有有限的技能格數，不同技能佔用不同格數（通常為 1 格）。玩家需要在有限的格數內選擇最適合的技能組合。

### 12.4 技能耗魔減少

部分裝備或特殊條件可以減少技能的魔力消耗：
- 格鬥士使用部分技能時，耗魔減半（如反擊、崩擊、明鏡止水）
- 傳教士使用治療技能時，耗魔減半
- 巫師使用輔助魔法時，耗魔減半

## 十四、設計參考建議

本節根據魔力寶貝原作系統，提供天命共振系統的設計參考。

### 14.1 屬性系統建議

魔力寶貝的五項基礎屬性（體力、力量、強度、速度、魔法）設計非常清晰，每項屬性都有明確的主要用途，但也有交叉影響（例如體力同時影響生命和回復，魔法同時影響魔力和精神）。這種設計讓加點有策略性，避免了「只加一種屬性」的無聊情況。

**建議：** 在天命共振系統中，可參考此設計，讓每項屬性都有 2-3 個影響面向，其中一個為主要影響，其餘為次要影響。

### 14.2 傷害計算建議

魔力寶貝的傷害公式採用「攻擊力 - 防禦力」的基礎結構，再乘以各種修正係數。這種設計的優點是直觀易懂，且各種修正（種族、屬性、技能倍率）可以疊加，創造豐富的戰術選擇。

**建議：** 保留目前的傷害結構，但可考慮加入「精神值」作為魔法攻擊的獨立計算軸，與物理攻擊的「攻擊力」分開計算。

### 14.3 技能分類建議

魔力寶貝的技能分類非常完整，特別是「吸收」、「反彈」、「無效」三種防禦性輔助魔法的設計，創造了豐富的戰術對抗空間：

- **吸收：** 將傷害轉為生命值（適合持久戰）
- **反彈：** 將傷害反彈給攻擊者（適合對抗強力攻擊）
- **無效：** 直接使傷害無效（最強但持續次數少）

**建議：** 在天命共振的技能圖鑑中，可加入這三種防禦機制作為技能類型選項。

### 14.4 全體技能的設計原則

魔力寶貝的全體技能通常有以下特點：
1. 耗魔更高（通常是單體技能的 1.5-2 倍）
2. 效果略弱（如超強補血的全體效果比補血魔法的單體效果低）
3. 需要更高等級才能解鎖全體範圍

**建議：** 在天命共振中，全體技能的傷害/治療量應比單體技能低 20-30%，以平衡遊戲性。

### 14.5 Boss 設計建議

根據魔力寶貝的 Boss 設計理念：
- Boss 通常擁有多種技能類型（物理、魔法、狀態異常）
- 部分 Boss 技能無法被反彈（如補血魔法不會被巫術反彈）
- Boss 的「每回合多次行動」設計可參考舞者技能的「跳舞」效果，讓 Boss 具有壓迫感
