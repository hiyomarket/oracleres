## 四、洞察力系統（密店 / 隱藏NPC / 隱藏任務）

### 4.1 洞察力定義

洞察力（`treasureHunting`，1-100）代表角色感知隱藏事物的能力。

**影響洞察力的因素：**

| 來源 | 效果 |
|------|------|
| 本命五行（水命） | 基礎洞察力 +15 |
| 流日天干（壬/癸水日） | 當日洞察力 +10 |
| 技能「流水感知」 | 洞察力 +10 |
| 靈相干預「神眼加持」 | 洞察力 ×1.15（10行動內） |
| 裝備詞條（感知類） | 洞察力 +5 至 +20 |
| 等級成長 | 每5級 +3 |

### 4.2 感知機率公式

**核心原則：所有人都有機會感知，洞察力只影響機率高低。**

```
感知機率 = min(95%, max(5%, (洞察力 / 感知門檻) × 100%))
```

- 洞察力 < 20：5%-15%（極低，但仍有機會）
- 洞察力 20-49：15%-50%（低至中等）
- 洞察力 50-79：50%-80%（中至高）
- 洞察力 ≥ 80：80%-95%（高，幾乎必然感知）

**感知門檻（perceptionThreshold）隨機範圍：**

| 事件類型 | 門檻範圍 | 說明 |
|---------|---------|------|
| 密店 | 20-50 | 相對容易感知 |
| 隱藏NPC | 25-55 | 中等難度 |
| 隱藏任務 | 30-60 | 較難感知 |
| 稀有密店 | 60-80 | 高洞察力才有較高機率 |

### 4.3 隨機放置邏輯

每次 Tick（約20分鐘）：
1. 清除過期事件
2. 若場上有效事件 < 10，有 40% 機率生成 1-2 個新事件
3. 新事件**隨機選取未佔用的地圖節點**放置
4. 事件存活 3-7 個 Tick（約1-2.5小時）

**重要**：事件不綁定特定節點，完全隨機，保持探索驚喜感。

### 4.4 洞察力 UI 呈現

- 密店按鈕顯示當前感知等級（極弱/微弱/偶有感知/感知強）
- 進入節點時若有隱藏事件，顯示「感知到異樣」的視覺提示
- 感知成功後，顯示事件詳情（密店商品/NPC對話/任務說明）

- **交易貨幣**：冒險幣（遊戲點）

- 靈石不可在拍賣行交易（防止點數套現）
