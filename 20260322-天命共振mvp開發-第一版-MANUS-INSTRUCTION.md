## 首頁架構

### 🏠 三大區塊

| 區塊 | 功能 | 價值 |
|------|------|------|
| **1. 錦囊體驗** | 輸入生日 → API 顯示今日運勢 | 立即感受價值 |
| **2. 功能卡片** | 命格身份證預覽、日曆、專家直通 | 展示服務 |
| **3. 社會認證** | 用戶見證文字、專家數量顯示 | 建立信任 |

### 🎮 遊戲化設計

- **核心玩法**：「今日運勢抽卡」
- **未登入用戶**：今日運勢抽卡 × 1次 + 部分功能預覽 + 分享按鈕
- **已登入用戶**：完整命盤查詢 + 元神系統 + 專屬推薦 + 每日抽卡次數解鎖
- **鉤子文案**：「登入後，你的元神將正式覺醒」

### 📊 轉化路徑

```
訪客抵達 → 輸入生日（好奇）→ 獲得價值（驚喜）→ 引導註冊 → 預約變現
```
