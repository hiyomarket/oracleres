## 🎯 執行階段規劃

### 第一階段：環境建置（Day 1-3）
- 1.1 購買 Vultr 4vCPU/8GB VPS
- 1.2 初始化 Ubuntu + 安裝 Docker
- 1.3 安裝 MySQL 8.0 + 設定帳密
- 1.4 申請網域 + Let's Encrypt SSL
- 1.5 設定 SSH 密鑰存取

### 第二階段：資料庫遷移（Day 4-5）
- 2.1 Clone hiyomarket/oracleres
- 2.2 建立 MySQL 資料庫
- 2.3 執行 full-dump.sql 還原
- 2.4 驗證 104 張資料表完整性
- 2.5 更新連線字串 (.env)

### 第三階段：系統部署（Day 6-8）
- 3.1 Clone oracle-mud 到 VPS
- 3.2 安裝依賴 (npm/pnpm install)
- 3.3 設定環境變數 (DB/S3/JWT Secret)
- 3.4 Build 前端 (npm run build)
- 3.5 設定 Nginx reverse proxy
- 3.6 設定 PM2 自動重啟

### 第四階段：功能驗證（Day 9-10）
- 4.1 會員登入/註冊功能測試
- 4.2 問卦/作戰室功能測試
- 4.3 遊戲數值/怪物圖鑑驗證
- 4.4 美術素材遷移測試

### 第五階段：上線（Day 11-14）
- 5.1 DNS 解析切換到新 VPS
- 5.2 HTTPS SSL 憑證設定
- 5.3 監控腳本部署 (CPU/記憶體)
- 5.4 備份自動化 (每日自動備份)
