## 📌 欄位說明（對應 DB-SCHEMA Monster DB）

| 欄位 | 說明 |
|:---|:---|
| monster_id | 系統流水號 |
| monster_code | 程式用 ID（如 M_F001）|
| 五行 | 火（固定）|
| 等級 | 1–60 |
| 稀有度 | 普通/精英/Boss/世界Boss |
| HP / 攻 / 防 / 速度 / 命中力 / 魔法攻擊 | 基礎屬性 |
| 抗木/火/土/金/水% | 五維抗性（0–90）|
| 被剋制加成% | 屬性被剋時額外承受傷害（0–50）|
| skill_id_1/2/3 | 怪物技能（關聯 MonsterSkill DB）|
| AI 等級 | 1–5 |
| 成長率 | 普通/良好/優秀/天賦/傳說 |
| 掉落素材 / 掉落機率% | 對應 Item DB |

## 六、戰鬥結算說明

### 6.1 火屬性剋制
```
火剋金 → 被剋制時：額外承受 (怪物被剋制加成% / 100) × 傷害
```

### 6.2 五維抗性
```
實際受到屬性傷害 = 技能基礎傷害 × (1 - 抗性% / 100)
抗性上限：90%
```
